# <a name="using-r-in-azure-sql-database"></a>Uso de R en la base de datos SQL Azure

En octubre de 2017, el equipo de desarrollo de SQL Server anunciado planes para admitir la ejecución de R código de bases de datos mediante procedimientos almacenados, similares a los servicios de R en SQL Server 2016.

> [!IMPORTANT]
> La versión preliminar inicial que se anunció se diseñó para pruebas y la exploración solo. Actualmente, la característica es **deshabilitado** en la base de datos de SQL de Azure para sustentar el desarrollo adicional. 

Para mantener al día en el público de la versión programación y próximos eventos, vea el [blog de SQL Server](https://blogs.technet.microsoft.com/dataplatforminsider/) o [blog de Microsoft R Server](https://blogs.msdn.microsoft.com/rserver/).

**Recursos de Azure**
