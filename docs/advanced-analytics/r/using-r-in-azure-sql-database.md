# <a name="using-r-in-azure-sql-database"></a>Uso de R en la base de datos SQL Azure

En octubre de 2017, el equipo de desarrollo de SQL Server anunciado planes para admitir la ejecución de R código de bases de datos mediante procedimientos almacenados, similares a los servicios de R en SQL Server 2016. Esta característica está en fase de desarrollo.

Para mantener al día en la programación de publicación y próximos eventos, vea el [blog de SQL Server](https://blogs.technet.microsoft.com/dataplatforminsider/) o [blog de Microsoft R Server](https://blogs.msdn.microsoft.com/rserver/).

> [!IMPORTANT]
> La versión preliminar inicial que se anunció se diseñó para pruebas y la exploración solo. Actualmente, la característica está disponible en la base de datos de SQL de Azure en regiones limitadas solo y características están limitadas en comparación con las características admitidas en SQL Server 2016 o de 2017.

**Recursos de Azure**

Mientras tanto, se recomienda que use una de las máquinas virtuales de SQL Server 2017 que están disponibles en el Marketplace de Azure:
