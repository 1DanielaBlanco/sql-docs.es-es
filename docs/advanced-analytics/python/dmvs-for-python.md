# <a name="dmvs-for-python"></a>DMV para Python

Esta sección está en desarrollo.

> [!NOTE]
> Compatibilidad con Python es una característica nueva de SQL Server 2017 y está en versión preliminar. Busque información más pronto. 

En general, puede supervisar los scripts externos, incluida una que se ejecutase Python, utilizando el mismo marco de trabajo que se proporcionó para la ejecución de script de R en SQL Server 2016. Para obtener más información, consulte [DMV para R Services](../r/dmvs-for-sql-server-r-services.md).
