# <a name="monitoring-python-in-sql-server"></a>Supervisión de Python en SQL Server
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]

Compatibilidad con Python es una característica nueva de SQL Server, disponible a partir de SQL Server 2017.

El marco de trabajo general para la seguridad de datos y la ejecución de script externo equivale a R. Puede supervisar los scripts externos, incluida una que se ejecutase Python, utilizando el mismo marco de trabajo que se proporcionó para la ejecución de script de R en SQL Server 2016. Para obtener más información, consulte [Monitoring R Solutions](../r/managing-and-monitoring-r-solutions.md).
