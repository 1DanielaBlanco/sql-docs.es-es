# <a name="monitoring"></a>Supervisión


> [!NOTE]
> Compatibilidad con Python es una característica nueva de SQL Server 2017 y está en versión preliminar.

Compatibilidad con Python es una característica nueva de SQL Server, disponible en SQL Server de 2017 CTP 2.0.

Sin embargo, el marco de trabajo general para la seguridad de datos y la ejecución de script externo equivale a R. Puede supervisar los scripts externos, incluida una que se ejecutase Python, utilizando el mismo marco de trabajo que se proporcionó para la ejecución de script de R en SQL Server 2016. Para obtener más información, consulte [Monitoring R Solutions](../r/managing-and-monitoring-r-solutions.md).
