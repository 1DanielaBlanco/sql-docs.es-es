# <a name="introducing-microsoft-technologies-for-data-storage-movement-and-transformation"></a>Introducción a las tecnologías de Microsoft para el almacenamiento, la transferencia y la transformación de datos
**Resumen:**el objetivo de este artículo es presentar las tecnologías de Microsoft para el almacenamiento, la transferencia y la transformación de datos. Estas tecnologías admiten la inteligencia empresarial mediante la habilitación de su organización para acceder a datos de calidad heterogéneos para análisis e informes, tanto en la nube como localmente.  
  
Para obtener información general sobre las tecnologías de inteligencia empresarial de Microsoft, consulte [Presentación de las herramientas de análisis e informes de Microsoft BI](https://msdn.microsoft.com/en-us/library/dn655131.aspx).
