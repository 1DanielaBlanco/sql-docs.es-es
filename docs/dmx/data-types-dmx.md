# <a name="data-types-dmx"></a>Tipos de datos (DMX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  Cuando usa extensiones de minería de datos (DMX) para definir un nuevo modelo de minería de datos en [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)], debe proporcionar un tipo de datos para cada columna en el modelo. El tipo de datos describe los datos que usará el algoritmo de minería de datos al generar el modelo.  
  
 Los tipos de datos dependen del algoritmo. Cada algoritmo determina los tipos de datos admitidos y su uso. Los algoritmos de Microsoft admiten los siguientes tipos de datos:  
  
-   Texto  
  
-   Long  
  
-   Boolean  
  
-   Doble  
  
-   date  
  
 Para obtener más información acerca de los datos de tipos que [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] admite, consulte [tipos de datos &#40;minería de datos&#41;](../analysis-services/data-mining/data-types-data-mining.md).
