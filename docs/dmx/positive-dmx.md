# <a name="-positive-dmx"></a>+ (Positivo) (DMX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  Realiza una operación unaria que devuelve el valor positivo de una expresión numérica.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
+ Numeric_Expression  
```  
  
#### <a name="parameters"></a>Parámetros  
 *Numeric_expression*  
 Expresión DMX (Extensiones de minería de datos) válida que devuelve un valor numérico.  
  
## <a name="return-value"></a>Valor devuelto  
 Valor cuyo tipo de datos es el del parámetro especificado.
