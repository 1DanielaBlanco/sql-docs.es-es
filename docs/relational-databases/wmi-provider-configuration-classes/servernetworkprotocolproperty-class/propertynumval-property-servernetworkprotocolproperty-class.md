# <a name="propertynumval-property-servernetworkprotocolproperty-class"></a>Propiedad PropertyNumVal (clase ServerNetworkProtocolProperty)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Obtiene el valor numérico de una propiedad que se hace referencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object.PropertyNumVal [= value]  
```  
  
## <a name="parts"></a>Partes  
 *objeto*  
 A [clase ServerNetworkProtocolProperty](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocolproperty-class/servernetworkprotocolproperty-class.md) objeto que representa un atributo del protocolo de red en la instancia de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad/Valor devuelto  
 Valor de **uint32** que especifica el valor numérico de la propiedad a la que se hace referencia.
