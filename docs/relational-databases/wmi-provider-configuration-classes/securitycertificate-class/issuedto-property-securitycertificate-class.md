# <a name="issuedto-property-securitycertificate-class"></a>Propiedad IssuedTo (clase SecurityCertificate)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Obtiene el objeto al que se emite el certificado de seguridad.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object.IssuedTo [= value]  
```  
  
## <a name="parts"></a>Partes  
 *objeto*  
 Objeto de la [clase SecurityCertificate](../../../relational-databases/wmi-provider-configuration-classes/securitycertificate-class/securitycertificate-class.md) que representa un certificado de seguridad.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el objeto al que se emite el certificado de seguridad.
