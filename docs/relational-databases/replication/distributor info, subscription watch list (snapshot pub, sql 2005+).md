# Informaci&#243;n de distribuidor, Lista de supervisi&#243;n de suscripciones (Publicaci&#243;n de instant&#225;neas, SQL Server 2005 y posteriores)
  El nombre del publicador se incluye en la información para las suscripciones de publicaciones de instantáneas. De lo contrario, la funcionalidad y la información proporcionadas en este cuadro de diálogo son las mismas que las de la vista Publicador. Para obtener más información acerca de cómo utilizar este cuadro de diálogo, vea [información del publicador, lista de observación de suscripción y Nº 40; Instantánea de publicación, SQL Server 2005 y versiones posterior y nº 41;](../../relational-databases/replication/publisher information, subscription watch list (snapshot).md).
