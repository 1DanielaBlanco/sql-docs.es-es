# Tipo de publicaci&#243;n
  La replicación proporciona los siguientes tipos de publicaciones:  
  
-   Publicación de instantáneas  
  
-   Publicación transaccional  
  
-   Publicación de combinación  
  
 El tipo o tipos de replicación que elija para su aplicación dependen del entorno físico de la replicación, el tipo y la cantidad de datos para los que desea crear una replicación, y de que los datos se actualicen en el suscriptor. El entorno físico incluye el número y la ubicación de los equipos que participan en la replicación, y si estos equipos son clientes (estaciones de trabajo, equipos portátiles o dispositivos de mano) o servidores. Para obtener más información, consulte [tipos de replicación](../../relational-databases/replication/types-of-replication.md).  
  
## Opciones  
 **Tipo de publicación**  
 Seleccione el tipo de replicación adecuado para esta publicación.
