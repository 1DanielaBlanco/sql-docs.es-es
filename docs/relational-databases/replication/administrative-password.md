# Contrase&#241;a administrativa
  If, en la **distribuidores** página de este asistente, seleccionó un distribuidor remoto de este publicador, debe escribir una contraseña para el realiza replicación de conexión entre el publicador y el distribuidor mediante la **distributor_admin** Inicio de sesión. La contraseña debe coincidir con la especificada en la página **Contraseña del distribuidor** del Asistente para configurar la distribución o en la página **Publicadores** del cuadro de diálogo **Propiedades del distribuidor** .  
  
## Opciones  
 **Contraseña**  
 Escriba la contraseña para la conexión entre el publicador y el distribuidor remoto.  
  
 **Confirmar contraseña**  
 Vuelva a escribir la contraseña para confirmar que es correcta.
