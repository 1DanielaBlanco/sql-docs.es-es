# <a name="sql-server-clr-object"></a>CLR (objeto de SQL Server)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] El objeto **SQLServer:CLR** proporciona contadores para supervisar la ejecución de Common Language Runtime (CLR) en [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
 En esta tabla se describen los contadores de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] **CLR** .  
  
|Contadores de CLR|Description|  
|------------------|-----------------|  
|**Ejecución de CLR**|Tiempo de ejecución total en CLR (en microsegundos)|
