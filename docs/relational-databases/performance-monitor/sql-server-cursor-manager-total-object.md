# <a name="sql-server-cursor-manager-total-object"></a>Cursor Manager Total (objeto de SQL Server)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] El objeto **SQLServer:Cursor Manager Total** proporciona contadores para supervisar cursores.  
  
 Esta tabla explica los contadores **Cursor Manager Total** de SQL Server.  
  
|Contadores Cursor Manager Total|Descripción|  
|-----------------------------------|-----------------|  
|**Recuento de llenado asincrónico**|Número de cursores que se llenan asincrónicamente.|  
|**Velocidad de conversión de cursores**|Número de conversiones de cursor por segundo.|  
|**Vaciados de cursor**|Número total de instrucciones que se crean de nuevo con cursores en tiempo de ejecución.|
