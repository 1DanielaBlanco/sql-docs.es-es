# <a name="ihindextypes-transact-sql"></a>IHindextypes (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  El **IHindextypes** tabla del sistema contiene una fila por cada tipo de índice no es de SQL Server admitido para publicadores servidor que no son de SQL. Esta tabla se almacena en la base de datos de distribución.  
  
|Nombre de columna|Tipo de datos|Description|  
|-----------------|---------------|-----------------|  
|**Tipo**|**nvarchar(255)**|El nombre de un tipo de índice admitido ajeno a SQL Server.|
