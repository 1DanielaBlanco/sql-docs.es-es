# Valor para propiedad extendida (cuadro de di&#225;logo)
  Use el cuadro de diálogo **Valor para** *<nombre de propiedad>\>* para especificar o ver un valor. Se trata de un cuadro de diálogo común que puede abrirse desde varias ubicaciones.  
  
## Lista de UIElement  
 **Nombre de propiedad extendida**  
 Nombre de la propiedad extendida que se muestra o se define.  
  
 **Valor de propiedad extendida**  
 Escriba o modifique el valor de la propiedad extendida.
