# <a name="input-and-output-error-during-hard-page-fault"></a>Input and Output Error During Hard Page Fault
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba que el identificador EventId 51 se encuentra en el registro de eventos del sistema. Este error se debe a otro error de página en disco.
