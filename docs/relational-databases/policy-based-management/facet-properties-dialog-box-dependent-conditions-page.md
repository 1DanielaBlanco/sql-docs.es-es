# <a name="facet-properties-dialog-box-dependent-conditions-page"></a>Cuadro de diálogo Propiedades de faceta, página Condiciones dependientes
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Use este cuadro de diálogo para mostrar una lista de las condiciones que hacen referencia actualmente a esta faceta de administración basada en directivas.  
  
## <a name="options"></a>Opciones  
 **Nombre**  
 Muestra el nombre de cada condición que hace referencia a la faceta actual.
