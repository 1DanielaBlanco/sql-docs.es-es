# <a name="device-not-ready-error"></a>Error por dispositivo no listo
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba que el identificador EventId 15 se encuentra en el registro de eventos del sistema. Este error puede ser debido a errores de configuración o problemas relacionados del adaptador host SCSI.
