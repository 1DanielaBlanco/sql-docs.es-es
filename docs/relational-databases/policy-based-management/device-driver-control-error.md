# <a name="device-driver-control-error"></a>Error de control del controlador de dispositivo
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba que el identificador EventId 11 se encuentra en el registro de eventos del sistema. Esto podría deberse a un controlador de dispositivo dañado, un problema de hardware, cableado defectuoso o a problemas de conectividad.
