# <a name="detect-failed-input-and-output-requests"></a>Detect Failed Input and Output Requests
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba que el identificador EventId 50 se encuentra en el registro de eventos del sistema. Este error se debe a una solicitud de E/S que no ha podido realizarse.
