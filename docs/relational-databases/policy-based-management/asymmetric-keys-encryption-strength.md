# <a name="asymmetric-keys-encryption-strength"></a>Intensidad del cifrado de claves asimétricas
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba si las claves asimétricas se crearon utilizando un cifrado de 1024 bits o más seguro.  
  
## <a name="best-practices-recommendations"></a>Prácticas recomendadas  
 Utilice RSA de 1024 bits o un cifrado más seguro para crear las claves asimétricas para el cifrado de los datos.
