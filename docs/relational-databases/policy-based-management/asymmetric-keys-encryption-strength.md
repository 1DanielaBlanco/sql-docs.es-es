# Intensidad del cifrado de claves asim&#233;tricas
  Esta regla comprueba si las claves asimétricas se crearon utilizando 1024 bits o un cifrado más seguro.  
  
## Prácticas recomendadas  
 Utilice RSA de 1024 bits o un cifrado más seguro para crear las claves asimétricas para el cifrado de los datos.
