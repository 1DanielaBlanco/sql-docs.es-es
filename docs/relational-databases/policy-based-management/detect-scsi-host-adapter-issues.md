# <a name="detect-scsi-host-adapter-issues"></a>Detectar problemas de adaptadores de host SCSI
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba que el identificador EventId 1066 se encuentra en el registro de eventos del sistema. Este error se debe a problemas de la configuración del adaptador de host SCSI o a un funcionamiento incorrecto del dispositivo.  
  
## <a name="best-practices-recommendations"></a>Prácticas recomendadas  
 Revise el artículo de Microsoft Knowledge Base siguiente para obtener más información sobre cómo solucionar este error:  
  
-   [Artículo 311081 de Microsoft Knowledge Base](http://go.microsoft.com/fwlink/?linkid=117744)
