# <a name="keep-the-affinity-mask-default-value"></a>Mantener el valor predeterminado de la máscara de afinidad
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Esta regla comprueba si el valor de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] correspondiente a la máscara de afinidad está establecido en 0, que es el valor predeterminado. La opción affinity mask controla dinámicamente la afinidad de la CPU.
