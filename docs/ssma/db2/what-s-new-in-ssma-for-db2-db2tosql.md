# <a name="what39s-new-in-ssma--for-db2-db2tosql"></a>¿Qué &#39; s de SSMA para DB2 (DB2ToSQL)
Este tema enumeran SSMA para cambios de DB2 en cada versión.  

## <a name="may-2016"></a>Mayo de 2016  
La versión de mayo de 2016 de SSMA para DB2 contiene los siguientes cambios:  

1.  Ha agregado compatibilidad con SQL Server 2016
2.  Conversión agregado de tablas de DB2 en memoria y regulares a las características de hekaton y en memoria de SQL Server
3.  Conversión agregado DB2 de controles de acceso a objetos de directiva de SQL Server (seguridad de nivel de fila para DB2)
4.  Conversión agregada de tablas con versión del sistema DB2 en tablas temporales de SQL Server
5.  Mejorar el analizador de DB2 y resolución
6.  Quita la comprobación de instalador para .net 2.0
7.  Quitar *.dll innecesarios de instalador de Db2
8.  Fija "guardar el proyecto" y "Abrir proyecto" comandos de consola SSMA
9.  Comando "securepassword" fijo para la consola de SSMA
10. Corregido el recuento de objetos para la carga inicial
11. Ha corregido el error en la configuración global
  
## <a name="march-2016"></a>Marzo de 2016  
La versión de vista previa de marzo de 2016 de SSMA para DB2 contiene los siguientes cambios:  
  
1.  Admite la migración a SQL Server 2016  
  
## <a name="january-2016"></a>Enero de 2016  
La versión de mantenimiento de enero de 2016 de SSMA para DB2 contiene los siguientes cambios:  
  
1.  Ha agregado compatibilidad para una serie de funciones estándares  
  
2.  Errores del analizador de DB2 fijo  
  
3.  Fijo DB2 v9 zOS soporte (RFC 5690920)  
  
4.  DB2 fijo sin resolver errores de identificador durante la conversión  
  
5.  Elemento de menú de registro de la vista agregada para SSMA (RFC 5706203)  
  
6.  Telemetría agregada  
  
## <a name="november-2014"></a>Noviembre de 2014  
La versión de noviembre de 2014 de SSMA para DB2 es el primero de su tipo.
