# <a name="welcome-to-sql-server-import-and-export-wizard"></a>Asistente para importación y exportación de SQL Server
  Al iniciar el Asistente para importación y exportación de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] , la primera página es **Asistente para importación y exportación de SQL Server**. No hace falta que realice ninguna acción en esta página.

> [!IMPORTANT]
> En este tema se describe solo la **primera página** del asistente. Si está buscando algo, consulte [relacionados con tareas y contenido](#related).

## <a name="prerequisite---is-the-wizard-installed-on-your-computer"></a>¿Requisito previo: es el Asistente para instalar en el equipo?
Si va a ejecutar el asistente, pero no tienes [! INCLUIR[msCoName](/sql-docs/docs/ssdt/download-sql-server-data-tools-ssdt).

## <a name="screen-shot-of-the-welcome-page"></a>Captura de pantalla de la página principal  
En la siguiente captura de pantalla se muestra la página **Asistente para importación y exportación de SQL Server** del asistente.  
  
![Página de bienvenida de la importación y el Asistente para exportación de](../../integration-services/import-export-data/media/welcome.png "página principal de la importación y el Asistente para exportación")  

## <a name="dont-show-this-page-again"></a>No volver a mostrar esta página  
**No volver a mostrar esta página.**  
 Omitir la página de bienvenida la próxima vez que ejecute al asistente.  
  
## <a name="whats-next"></a>¿Qué debe hacer a continuación?  
 Después de la página principal aparece la página **Seleccionar un origen de datos**. En esta página, debe proporcionar información sobre el origen de datos y la manera de conectarse a él. Para obtener más información, vea [Elegir un origen de datos](../../integration-services/import-export-data/choose-a-data-source-sql-server-import-and-export-wizard.md).
