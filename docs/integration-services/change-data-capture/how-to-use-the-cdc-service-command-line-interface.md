# C&#243;mo usar la interfaz de l&#237;nea de comandos del servicio CDC
  El programa del servicio CDC de Oracle, xdbcdcsvc.exe, ejecuta normalmente el servicio de Windows CDC de Oracle pero se puede invocar directamente desde la línea de comandos para crear o eliminar un servicio de Windows CDC de Oracle.  
  
### Para usar la línea de comandos  
  
1.  En el menú **Inicio**, abra la consola de la línea de comandos. Escriba **cmd** en el cuadro Ejecutar o Buscar para abrir la consola.  
  
2.  En el símbolo del sistema, escriba **cd** y la ruta de acceso completa a la carpeta donde está instalado el servicio CDC para Oracle.  
  
3.  Escriba el comando necesario para realizar una de las tareas posibles.
