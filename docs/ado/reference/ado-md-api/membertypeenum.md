# <a name="membertypeenum"></a>MemberTypeEnum
Especifica la configuración para la [tipo](../../../ado/reference/ado-md-api/type-property-ado-md.md) propiedad de un [miembro](../../../ado/reference/ado-md-api/member-object-ado-md.md) objeto.  
  
|Constante|Valor|Descripción|  
|--------------|-----------|-----------------|  
|**adMemberAll**|4|Indica que la **miembro** representa todos los miembros del nivel de objeto.|  
|**adMemberFormula**|3|Indica que la **miembro** objeto se calcula utilizando una expresión de una fórmula.|  
|**adMemberMeasure**|2|Indica que la **miembro** objeto pertenece a la dimensión Measures y que representa un atributo cuantitativo.|  
|**adMemberRegular**|1|Predeterminado: Indica que la **miembro** objeto representa una instancia de una entidad de negocio.|  
|**adMemberUnknown**|0|No se puede determinar el tipo del miembro.|
