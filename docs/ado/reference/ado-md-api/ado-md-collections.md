# <a name="ado-md-collections"></a>Colecciones de ADO MD
|||  
|-|-|  
|[Ejes](../../../ado/reference/ado-md-api/axes-collection-ado-md.md)|Contiene el **eje** objetos que definen un conjunto de celdas.|  
|[CubeDefs](../../../ado/reference/ado-md-api/cubedef-object-ado-md.md)|Contiene el **CubeDef** objetos que representan un cubo de un catálogo multidimensional.|  
|[Dimensiones](../../../ado/reference/ado-md-api/dimension-object-ado-md.md)|Contiene el **dimensión** objetos que componen un cubo.|  
|[Jerarquías](../../../ado/reference/ado-md-api/hierarchy-object-ado-md.md)|Contiene el conjunto **jerarquía** objetos de una dimensión.|  
|[Levels](../../../ado/reference/ado-md-api/level-object-ado-md.md)|Contiene el **nivel** objetos que componen una jerarquía.|  
|[Miembros](../../../ado/reference/ado-md-api/members-collection-ado-md.md)|Contiene el **miembro** objetos de un nivel o una posición a lo largo de un eje.|  
|[Posiciones](../../../ado/reference/ado-md-api/positions-collection-ado-md.md)|Contiene el **posición** objetos que definen un punto en un eje.|
