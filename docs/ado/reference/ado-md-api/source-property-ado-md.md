# <a name="source-property-ado-md"></a>Propiedad de origen (ADO MD)
Indica el origen de datos en el [cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Configuración y valores devueltos  
 Establece o devuelve un **Variant**, y es de lectura/escritura de cerrará [Cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) objetos y de solo lectura para abrir **Cellset** objetos. El **Variant** debe contener válido **cadena**, por ejemplo, una consulta MDX.
