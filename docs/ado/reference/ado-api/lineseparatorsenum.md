# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
Especifica el carácter utilizado como separador de línea en texto [flujo](../../../ado/reference/ado-api/stream-object-ado.md) objetos.  
  
|Constante|Value|Description|  
|--------------|-----------|-----------------|  
|**adCR**|13|Indica un retorno de carro.|  
|**adCRLF**|-1|Predeterminado: Indica la línea de retorno de carro, avance.|  
|**adLF**|10|Indica el avance de línea.|  
  
## <a name="adowfc-equivalent"></a>Equivalente ADO/WFC  
 Estas constantes no tienen equivalentes ADO/WFC.
