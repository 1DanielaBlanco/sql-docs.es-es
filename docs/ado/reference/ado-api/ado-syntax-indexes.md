# <a name="ado-syntax-indexes"></a>Índices de sintaxis de ADO
La sintaxis para llamar a métodos y propiedades ADO varía según el entorno de desarrollo. El resto de la referencia del lenguaje ADO utiliza el lenguaje de programación Microsoft Visual Basic para ilustrar la sintaxis de método y propiedad de ADO. Sin embargo, consulte las secciones siguientes para obtener ejemplos más específicos sintaxis según el lenguaje de programación y la misma metodología:  
  
-   El [ADO para el índice de sintaxis de Visual C++ para COM](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-for-com.md) cubre los métodos y propiedades de ADO sin utilizar la **#import** directiva de compilador con Microsoft Visual C++.  
  
-   El [ADO para el índice de sintaxis de Visual C++ con #import](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-with-sharpimport.md) trata ADO propiedades y métodos cuando se utiliza el **#import** directiva de compilador con Microsoft Visual C++.
