# <a name="filter-traces-with-sql-server-profiler"></a>Filtrar seguimientos con SQL Server Profiler
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]Puede definir filtros en columnas de datos para [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)] eventos de seguimiento para que se captura información únicamente de los eventos que interesa. Los filtros de seguimiento reducen el impacto de supervisar el rendimiento del sistema porque se capturan menos eventos en el seguimiento. Los filtros también facilitan la lectura de la salida porque hay que buscar en menos eventos.
