# Quitar un servidor registrado (SQL Server Management Studio)
  En este tema se describe cómo quitar un servidor de Servidores registrados en [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] mediante [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)].  
  
##  <a name="SSMSProcedure"></a>  
  
#### Para quitar un servidor registrado  
  
1.  En Servidores registrados, haga clic con el botón derecho en un servidor y, luego, haga clic en **Eliminar**.  
  
2.  En el cuadro de diálogo **Confirmación de eliminación** , haga clic en **Sí**.
