# Usar comentarios
[!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] permite comentar fácilmente parte de un script.  
  
## Comentar parte de un script  
  
#### Para comentar parte de un script  
  
1.  Utilice el mouse (ratón) para seleccionar el texto `WHERE LastName = 'Sanchez'`.  
  
2.  En el menú **Editar** , señale **Avanzadas**y, a continuación, haga clic en **Selección con comentarios**. El texto seleccionado como comentario se acota entre guiones (--).  
  
Además de utilizar el menú **Editar** , puede acotar como comentario una parte seleccionada del script o quitar el comentario utilizando cualquiera de los métodos siguientes:  
  
-   Haga clic en los botones de la barra de herramientas del Editor SQL.  
  
-   Presione CTRL+K y CTRL+C para acotar como comentario el texto o presione CTRL+K, CTRL+U para quitar el comentario del texto.
