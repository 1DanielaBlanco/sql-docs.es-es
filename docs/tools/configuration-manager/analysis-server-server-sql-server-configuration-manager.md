# Analysis Server &lt;servidor&gt; (Administrador de configuraci&#243;n de SQL Server)
  Éste es el servicio que admite Microsoft [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)]. Para ver las propiedades del servicio, en el panel de detalles, haga clic con el botón derecho en el servicio y, luego, haga clic en **Propiedades**.  
  
 Para obtener información acerca de [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)], vea los Libros en pantalla.
