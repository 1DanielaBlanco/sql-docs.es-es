# Notification Services (Administrador de configuraci&#243;n de SQL Server)
  Este es el servicio que admite [!INCLUDE[ssNS](../../includes/ssns-md.md)]. Para ver las propiedades del servicio, en el panel de detalles, haga clic con el botón derecho en el servicio y, luego, haga clic en **Propiedades**.  
  
 Para obtener información acerca de [!INCLUDE[ssNS](../../includes/ssns-md.md)], vea los Libros en pantalla.
