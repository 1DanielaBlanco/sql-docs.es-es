# <a name="group-by-expression-list"></a>GROUP BY-lista de expresiones
GROUP BY es compatible con una lista de expresiones, así como un nombre de columna. La lista de selección de una instrucción SELECT que tenga una cláusula GROUP BY solo puede incluir expresiones de la cláusula GROUP BY o funciones del conjunto.
