# <a name="installing-the-software-odbc"></a>Instalar el Software (ODBC)
> [!IMPORTANT]  
>  Esta característica se quitará en una versión futura de Windows. Evite utilizar esta característica en nuevos trabajos de desarrollo y tenga previsto modificar las aplicaciones que actualmente la utilizan. En su lugar, utilice el controlador ODBC proporcionado por Oracle.  
  
 El controlador ODBC para Oracle es uno de los componentes de acceso a datos. Acompaña a otros componentes ODBC, como el Administrador de orígenes de datos ODBC y deben estar ya instalado. El controlador también se puede encontrar en "Controladores y otras descargas" en el sitio Web de servicios de soporte técnico de producto de Microsoft Online en www.microsoft.com.  
  
 Debe instalarse el software de red según su propia documentación. El controlador ODBC para Oracle no requiere ninguna consideración especial instalación siempre que el software de red es compatible.  
  
 Según su propia documentación, se debe instalar software de Oracle. El controlador ODBC para Oracle no requiere normalmente ninguna consideración especial instalación siempre y cuando el controlador es compatible con la versión. Sin embargo, para mantener los productos compatibles, instalar al controlador ODBC para Oracle última para asegurarse de que tiene la última versión del controlador. Oracle mantiene un sitio FTP público que expone, entre otras cosas, revisiones de los productos de servidor de Oracle y el componente de cliente que se incluye con los productos de servidor. Estas revisiones son necesarias para que funcione correctamente varios productos y tecnologías de Microsoft. Para obtener más información acerca de este sitio, consulte [revisiones de Software de Oracle](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  La instalación de software de Oracle a través de MDAC/Windows DAC, puede sobrescribir las versiones actuales de MDAC. Si surgen problemas con componentes de ODBC, vuelva a instalar MDAC.
