# <a name="visual-foxpro-terminology"></a>Terminología de Visual FoxPro
**database**  
 En Visual FoxPro, un archivo de base de datos tiene una extensión de .dbc y puede contener uno o varios **tablas**.  
  
 **tabla de base de datos**  
 En Visual FoxPro, una tabla que está asociado a una base de datos. Contraste **tabla libre**.  
  
 **tabla libre**  
 En Visual FoxPro, una tabla que no está asociada a una base de datos.  
  
 Un archivo .dbf creado en la versión de FoxPro 2.x es una tabla disponible a menos que se convierte en una tabla de Visual FoxPro y se agrega a una base de datos de Visual FoxPro. Contraste **tabla de base de datos**.  
  
 **instrucción SQL puede**  
 Una instrucción SQL que ya no se ha procesado por el **SQLPrepare** función. Para obtener más información acerca de esta función con el controlador ODBC de Visual FoxPro, consulte [SQLPrepare (controlador de ODBC de Visual FoxPro)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 **table**  
 En Visual FoxPro, los registros se almacenan en una tabla. Cada fila de una tabla representa un registro y las columnas de la tabla enumeran los campos del registro. Cada tabla de Visual FoxPro se almacena en su propio archivo con una extensión .dbf. Tablas de Visual FoxPro pueden asociarse con una base de datos.  
  
 FoxPro versión 2. *x* tablas no están asociadas a una base de datos.
