# <a name="error-messages-odbc-driver-for-oracle"></a>Mensajes de error (controlador ODBC para Oracle)
> [!IMPORTANT]  
>  Esta característica se quitará en una versión futura de Windows. Evite utilizar esta característica en nuevos trabajos de desarrollo y tenga previsto modificar las aplicaciones que actualmente la utilizan. En su lugar, utilice el controlador ODBC proporcionado por Oracle.  
  
 Cuando se produce un error, el controlador ODBC de Microsoft para Oracle devuelve el valor de SQLSTATE (un código de error ODBC) y un mensaje de error. El controlador deriva esta información tanto de los errores detectados por el controlador de errores devueltos por el servidor de Oracle.
