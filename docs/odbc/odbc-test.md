# <a name="odbc-test"></a>Prueba ODBC
Prueba de ODBC de Microsoft® es una aplicación habilitada para ODBC que puede utilizar para probar controladores ODBC y el Administrador de controladores ODBC. ODBC 3.51 incluye ANSI y habilitada para Unicode versiones de prueba de ODBC. Los archivos correspondientes son los siguientes:  
  
-   Odbcte32.exe y Gtrtst32.dll, para la versión ANSI.  
  
-   Odbct32w.exe y Gtrts32w.dll, para la versión de Unicode.  
  
 Para usar pruebas de ODBC, debe entender la API de ODBC, el lenguaje C y SQL. Para obtener más información acerca de la API de ODBC, consulte el [referencia del programador de ODBC](../odbc/reference/odbc-programmer-s-reference.md).  
  
 Temas de ayuda que se incluye anteriormente en esta sección de la documentación ahora están contenidas en el programa de prueba de ODBC. Abra Odbcte32.exe o Odbct32w.exe, abra el **ayuda** menú y, a continuación, haga clic en **temas de Ayuda**.  
  
 Tenga en cuenta que las versiones de 64 bits de estas aplicaciones, destinadas a sistemas de operativos de Microsoft Windows de 64 bits, tienen los mismos nombres que las versiones de 32 bits, aunque son archivos independientes. es decir, el nombre de la versión Unicode de la versión de 64 bits de prueba de ODBC es odbct32w.exe.
