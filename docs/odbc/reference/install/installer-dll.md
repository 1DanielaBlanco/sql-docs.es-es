# <a name="installer-dll"></a>Instalador de DLL
> [!NOTE]  
>  A partir de Windows XP y Windows Server 2003, ODBC se incluye en el sistema operativo de Windows. Solo explícitamente debe instalar ODBC en versiones anteriores de Windows.  
  
 El archivo DLL del instalador contiene funciones para instalar y quitar componentes ODBC, mantener la información del registro acerca de los componentes y mantener la información del registro acerca de los orígenes de datos. Está escrito por Microsoft y se puede redistribuir. Para obtener una descripción completa de las funciones en el archivo DLL del programa de instalación, consulte [referencia API DLL del instalador](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
