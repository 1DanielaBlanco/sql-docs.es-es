# <a name="odbc-header-files"></a>Archivos de encabezado de ODBC
> [!NOTE]  
>  La redistribución de archivos de encabezado ODBC no es necesaria. No se consideran "Core".  
  
 **Archivos de encabezado**  
  
 Odbcinst.h  
  
 SQL.h  
  
 Sqlext.h  
  
 SqlTypes.h  
  
 Sqlucode.h  
  
 MSDASQL.h  
  
 Msdadc.h  
  
 Sqlspi.h (se usa para el desarrollo de controladores, consulte [referencia de la interfaz de proveedor de servicios (SPI) de ODBC](../../../odbc/reference/syntax/odbc-service-provider-interface-spi-reference.md) para obtener más información) disponible a partir del SDK de Windows 8.
