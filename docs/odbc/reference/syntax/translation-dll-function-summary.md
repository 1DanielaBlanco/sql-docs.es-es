# <a name="translation-dll-function-summary"></a>Resumen de funciones del archivo DLL de traducción
En la tabla siguiente se describe las funciones del archivo DLL de traducción. Para obtener más información sobre la sintaxis y semántica para cada función, consulte [referencia de funciones de DLL de traducción](../../../odbc/reference/syntax/translation-dll-api-reference.md).  
  
|Tarea|Nombre de función|Finalidad|  
|----------|-------------------|-------------|  
|Traducir los datos|[SQLDataSourceToDriver](../../../odbc/reference/syntax/sqldatasourcetodriver-function.md)|Convierte todos los datos fluyen desde el origen de datos para el controlador.|  
||[SQLDriverToDataSource](../../../odbc/reference/syntax/sqldrivertodatasource-function.md)|Convierte todos los datos fluyen desde el controlador al origen de datos.|
