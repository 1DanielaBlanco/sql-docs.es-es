# <a name="sql-to-c-character"></a>SQL a caracteres C:
Los identificadores para los tipos de datos SQL de ODBC de caracteres son:  
  
 SQL_CHAR  
  
 SQL_VARCHAR  
  
 SQL_LONGVARCHAR  
  
 SQL_WCHAR  
  
 SQL_WVARCHAR  
  
 SQL_WLONGVARCHAR
