# <a name="job-properties---new-job-alerts-page"></a>Propiedades del trabajo - Nuevo trabajo (página Alertas)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Use esta página para ver y cambiar las alertas de un trabajo del Agente [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)].  
  
## <a name="options"></a>.  
**Lista de alertas**  
Presenta una lista de las alertas de este trabajo.  
  
**Agregar**  
Crea una nueva alerta para el trabajo.  
  
**Editar**  
Modifica la definición de la alerta seleccionada.  
  
**Quitar**  
Quita la alerta seleccionada del trabajo y la elimina.
