# <a name="proxy-account-properties-references-tab"></a>Propiedades de proxy (pestaña Referencias)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] En esta página de solo lectura se enumeran los pasos del trabajo que hacen referencia a una cuenta de proxy del Agente [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)].  
  
## <a name="options"></a>.  
**Nombre del trabajo**  
Nombres de los trabajos que usan esta cuenta de proxy.  
  
**Paso de trabajo**  
Nombres de los pasos del trabajo que usa esta cuenta de proxy.  
  
**Subsistema**  
Subsistema que usa esta cuenta de proxy.
