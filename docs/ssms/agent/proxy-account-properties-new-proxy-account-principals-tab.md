# <a name="proxy-account-properties---new-proxy-account-principals-tab"></a>Propiedades de cuenta de proxy - Nueva cuenta de proxy (pestaña Entidades de seguridad)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Use esta página para ver o cambiar las entidades de seguridad que pueden usar una cuenta de proxy del Agente [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] en los pasos de trabajo.  
  
## <a name="options"></a>.  
**Entidades de seguridad de la cuenta de proxy**  
Muestra las entidades de seguridad que pueden utilizar esta cuenta de proxy.  
  
**Agregar**  
Agrega una entidad de seguridad a la lista.  
  
**Quitar**  
Quita la entidad de seguridad seleccionada de la lista.
