# <a name="autosize-selected-tables-in-diagrams-visual-database-tools"></a>Ajustar automáticamente el tamaño de las tablas seleccionadas en diagramas (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Las tablas del diagrama se pueden ajustar rápidamente para que adopten el tamaño más pequeño posible de modo que se muestren todos los campos.  
  
### <a name="to-automatically-size-selected-tables-in-diagrams"></a>Para cambiar automáticamente el tamaño de las tablas seleccionadas en los diagramas  
  
1.  Seleccione la tabla o las tablas cuyo tamaño desea cambiar.  
  
2.  Haga clic con el botón derecho en una de las tablas seleccionadas y, en el menú contextual, elija **Ajustar automáticamente tablas seleccionadas**.
