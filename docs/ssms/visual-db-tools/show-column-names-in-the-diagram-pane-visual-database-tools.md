# <a name="show-column-names-in-the-diagram-pane-visual-database-tools"></a>Mostrar nombres de columna en el panel Diagrama (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Puede decidir si quiere ocultar o mostrar los nombres de columna de las tablas en el **panel Diagrama** del Diseñador de consultas y vistas.  
  
### <a name="show-column-names"></a>Mostrar nombres de columna  
  
1.  Haga clic con el botón derecho en la tabla del **panel Diagrama** del Diseñador de consultas y vistas.  
  
2.  Elija **Nombres de columna** en el menú contextual.  
  
### <a name="hide-column-names"></a>Ocultar nombres de columna  
  
1.  Haga clic con el botón derecho en la tabla del **panel Diagrama** del Diseñador de consultas y vistas.  
  
2.  Elija **Solo nombre** en el menú contextual.
