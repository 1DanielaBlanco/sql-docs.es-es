# <a name="show-query-properties-visual-database-tools"></a>Mostrar propiedades de la consulta (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Puede mostrar las propiedades de una consulta en la ventana Propiedades desde el Diseñador de consultas y vistas.  
  
### <a name="to-display-properties-for-a-query"></a>Para mostrar las propiedades de una consulta  
  
-   Con una consulta abierta en el Diseñador de consultas y vistas, en el menú **Ver** , haga clic en la ventana **Propiedades**.  
  
Podrá ver las propiedades de la consulta en la ventana **Propiedades** .
