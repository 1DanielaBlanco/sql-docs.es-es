# <a name="view-join-properties-visual-database-tools"></a>Ver las propiedades de una combinación (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
En la ventana Propiedades, puede ver una colección de propiedades de una vista.  
  
### <a name="to-view-join-properties"></a>Para ver las propiedades de una combinación  
  
1.  En el panel **Diagrama** , haga clic con el botón derecho en la combinación.  
  
2.  En el menú contextual, haga clic en **Propiedades**.  
  
    Las propiedades de la combinación aparecerán en la ventana Propiedades.
