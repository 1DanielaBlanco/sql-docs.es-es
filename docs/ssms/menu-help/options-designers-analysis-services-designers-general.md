# <a name="options-designers---analysis-services-designers---general"></a>Opciones (Diseñadores - Diseñadores de Analysis Services - General)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Use la página **Diseñadores**, **Planes de mantenimiento**, **Analysis Services**, **General** para determinar el comportamiento predeterminado de los Diseñadores de Analysis Services.  
  
## <a name="connectivity"></a>Conectividad  
Tiempo de espera de la consulta  
Número de segundos que espera el diseñador a que responda la consulta antes de generar un error.  
  
Tiempo de espera de la conexión  
Número de segundos que espera el diseñador a que se confirme una conexión antes de generar un error.
