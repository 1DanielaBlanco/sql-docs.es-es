# <a name="attribute-properties---bind-an-attribute-to-a-key-column"></a>Propiedades de atributo: enlazar un atributo a una columna de clave
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  Este procedimiento describe cómo cambiar la configuración de la propiedad **Name** de un atributo en una dimensión.  
  
### <a name="to-bind-an-attribute-to-a-key-column"></a>Para enlazar un atributo a una columna de clave  
  
1.  En el Diseñador de dimensiones, abra la dimensión que contiene el atributo cuyo nombre desea cambiar.  
  
2.  En el panel **Atributos** de la vista Generador de dimensiones, cambie el formato de visualización a **Árbol** o **Lista**. Para obtener más información sobre cómo cambiar el formato del panel Atributos, vea [Ver los atributos de un árbol, una lista o una cuadrícula en el Diseñador de dimensiones](../../analysis-services/multidimensional-models/view-attributes-in-dimension-designer.md).  
  
3.  Haga clic con el botón derecho en el atributo que quiera configurar y, después, haga clic en **Cambiar nombre**.  
  
4.  Escriba el nuevo nombre.  
  
    > [!NOTE]  
    >  También se puede establecer la propiedad **Name** de un atributo seleccionado en la ventana **Propiedades de** [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)].
