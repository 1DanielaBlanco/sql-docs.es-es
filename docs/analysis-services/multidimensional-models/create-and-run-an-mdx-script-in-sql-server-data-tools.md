# Crear y ejecutar un script MDX en Herramientas de datos de SQL Server
  Para crear y ejecutar un script MDX en  [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)], debe estar en [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)] con un cubo ya creado y listo para editar.  
  
### Para crear un script MDX (Expresiones multidimensionales)  
  
1.  Abra el cubo para el que desea crear un script MDX, y en **Vista**, haga clic en **Cálculos**.  
  
2.  Si no está en modo **Vista de script** , haga clic en el icono **Vista de script** .  
  
3.  En la ventana de script, bajo el comando CALCULATE, escriba una expresión MDX. Haga clic en el icono **Comprobar sintaxis** y compruebe que la expresión es sintácticamente correcta.  
  
4.  Para ejecutar el script MDX, implemente y procese el cubo con los nuevos cambios de script MDX.
