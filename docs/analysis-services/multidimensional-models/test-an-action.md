# <a name="test-an-action"></a>Probar una acción
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]Para probar una acción mediante la vista de explorador en el Diseñador de cubos. Se puede obtener acceso al Diseñador de cubos desde [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]. Una vez creada la acción, debe procesar el cubo para poder probarla. Para más información, vea [Procesar un modelo multidimensional &#40;Analysis Services&#41;](../../analysis-services/multidimensional-models/processing-a-multidimensional-model-analysis-services.md).  
  
### <a name="to-test-an-action"></a>Para probar una acción  
  
1.  Abra el cubo que contiene la acción que desea probar y, en **Ver**, haga clic en **Explorador**.  
  
2.  Arrastre el objeto de destino a **Coloque campos de columna aquí** o a **Coloque campos de fila aquí**.  
  
3.  En el panel de vista, haga clic con el botón secundario en el objeto al que está adjuntada la acción (por ejemplo, una celda). El nombre de la acción aparecerá en el menú contextual.  
  
4.  Haga clic en el nombre de la acción para probarla.
