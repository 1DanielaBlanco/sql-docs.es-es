# <a name="propertylist-element---supported-xmla-properties"></a>Elemento PropertyList - admiten propiedades XMLA

Utilice estas propiedades enumeradas en la [propiedades](../../../analysis-services/xmla/xml-elements-properties/properties-element-xmla.md) elemento de la [Discover](../../../analysis-services/xmla/xml-elements-methods-discover.md) y [Execute](../../../analysis-services/xmla/xml-elements-methods-execute.md) métodos.
