# <a name="session-events-event-category"></a>eventos de sesión, categoría de eventos
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]La categoría de eventos de eventos de sesión tiene las clases de eventos se describen en la tabla siguiente.  
  
|Clase de eventos|Identificador del evento|Description|  
|-----------------|--------------|-----------------|  
|Existing Connection|41|Recopila todos los eventos de conexión desde el inicio del seguimiento.|  
|Existing Session|42|Recopila todos los eventos de sesión desde el inicio del seguimiento.|  
|Session Initialize|43|Recopila los eventos de inicialización de sesión.|  
  
 Para obtener información sobre las columnas asociadas a cada una de las clases de evento de Eventos de sesión, vea [Session Events Data Columns](../../analysis-services/trace-events/session-events-data-columns.md).
