# <a name="appliance-topology"></a>Topología de dispositivos
Muestra información resumida de todo el sistema de plataforma de análisis.  
  
Expanda los elementos de esta sección para configurar la información que se aplica al dispositivo completo. Seleccione las secciones de topología de almacenamiento de datos paralelos o la topología de HDInsight para configurar la información que se aplica sólo de esa región del dispositivo.
