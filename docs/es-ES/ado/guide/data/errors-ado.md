# <a name="errors-ado"></a>Errores (ADO)
Cualquier operación que implique objetos ADO puede generar uno o más errores del proveedor. Cuando se produce cada error, uno o varios **Error** objetos se colocan en la **errores** colección de la **conexión** objeto. Para obtener más información sobre el control de advertencias y errores en una aplicación de ADO, vea [control de errores](../../../ado/guide/data/error-handling.md).  
  
 Errores de aplicación se pueden generar un mecanismo independiente. Por ejemplo, en Visual Basic, la **Err** objeto contendrá errores de nivel de aplicación.
