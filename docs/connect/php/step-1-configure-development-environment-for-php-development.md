# <a name="step-1-configure-environment-for-php-development"></a>Paso 1: Configurar el entorno de desarrollo de PHP
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* Identificar qué versión del controlador PHP va a usar, en función del entorno, como se indica aquí: [requisitos del sistema para Drivers de Microsoft para PHP para SQL Server](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* Descargar e instalar controladores de PHP aplicables aquí: [descargar controladores de PHP de Microsoft](https://www.microsoft.com/download/details.aspx?id=20098)  
* Descargar e instalar controladores ODBC aplicables aquí: [descargar el controlador de ODBC para SQL Server](../../connect/odbc/download-odbc-driver-for-sql-server.md)  
* Configure los controladores de PHP y el servidor web para su sistema operativo específico:

### <a name="windows"></a>Windows  
  

* Configurar la carga del controlador PHP, como se indica aquí: [Loading the Microsoft Drivers for PHP para SQL Server](../../connect/php/loading-the-php-sql-driver.md) 
* Configurar IIS para hospedar aplicaciones PHP, como se indica aquí: [configurar IIS para Drivers de Microsoft para PHP para SQL Server](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux y Mac


*   Configurar la carga del controlador PHP y configurar el servidor web para hospedar aplicaciones PHP, como se indica a continuación: [Tutorial de instalación de controladores de Mac y Linux de PHP](../../connect/php/installation-tutorial-linux-mac.md)
