# <a name="converting-data-types"></a>Convertir tipos de datos
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

Los [!INCLUDE[ssDriverPHP](../../includes/ssdriverphp_md.md)] permiten especificar los tipos de datos al enviar o recuperar datos de [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]. No es obligatorio especificar los tipos de datos. Si no se especifican los tipos de datos, se utilizan tipos de valor predeterminado. En los temas de esta sección se describe cómo especificar tipos de datos y proporcionar información sobre los tipos de datos predeterminados.

|[Tipos de datos PHP predeterminados](../../connect/php/default-php-data-types.md)|Proporciona información sobre los tipos de datos PHP predeterminados al recuperar datos del servidor.|
