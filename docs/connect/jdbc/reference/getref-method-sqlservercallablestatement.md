# <a name="getref-method-sqlservercallablestatement"></a>Método getRef (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un objeto de referencia en el lenguaje de programación Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Description|  
|----------|-----------------|  
|[getRef (int)](../../../connect/jdbc/reference/getref-method-int.md)|Recupera el valor del parámetro designado como un objeto de referencia en el lenguaje de programación Java según el índice del parámetro.|  
|[getRef (java.lang.String)](../../../connect/jdbc/reference/getref-method-java-lang-string.md)|Recupera el valor del parámetro designado como un objeto de referencia en el lenguaje de programación Java según el nombre del parámetro.|
