# <a name="getdrivername-method-sqlserverdatabasemetadata"></a>Método getDriverName (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el nombre de este controlador JDBC.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.lang.String getDriverName()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 A **cadena** que contiene el nombre del controlador JDBC.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getDriverName especificado por el método getDriverName en la interfaz java.sql.DatabaseMetaData.
