# <a name="setarray-method-sqlserverpreparedstatement"></a>setArray (método) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el número de parámetro designado para el objeto de matriz determinado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final void setArray(int i,  
                           java.sql.Array x)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *i*  
  
 Un **int** que indica el número de parámetro.  
  
 *x*  
  
 Un objeto de matriz.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método setArray es especificado por el método setArray en la interfaz java.sql.PreparedStatement.
