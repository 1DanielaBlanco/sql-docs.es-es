# <a name="getinstancename-method-sqlserverdatasource"></a>Método getInstanceName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve el [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] nombre de instancia.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.lang.String getInstanceName()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 A **cadena** que contiene la instancia del nombre, o null si no se establece ningún valor.
