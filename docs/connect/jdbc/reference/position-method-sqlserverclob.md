# <a name="position-method-sqlserverclob"></a>Método position (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera la posición de caracteres del objeto CLOB especificado o la subcadena en el CLOB según la posición de inicio especificada.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Description|  
|----------|-----------------|  
|[posición (java.sql.Clob, long)](../../../connect/jdbc/reference/position-method-java-sql-clob-long.md)|Devuelve la posición de un carácter del objeto CLOB especificado en el CLOB según la posición de inicio determinada.|  
|[posición (java.lang.String, long)](../../../connect/jdbc/reference/position-method-java-lang-string-long.md)|Devuelve la posición de un carácter de la subcadena especificada en el CLOB según la posición de inicio determinada.|
