# <a name="getresultsetholdability-method-sqlserverstatement"></a>getResultSetHoldability (método) (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el conjunto de resultados capacidad de alojamiento para [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objetos que haya generado por este [SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md) objeto.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final int getResultSetHoldability()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un **int** que indica el conjunto de resultados capacidad de alojamiento.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getResultSetHoldability especificado por el método getResultSetHoldability en la interfaz java.sql.Statement.
