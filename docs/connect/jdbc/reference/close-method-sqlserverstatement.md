# <a name="close-method-sqlserverstatement"></a>Close (método) (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Esto libera [SQLServerStatement](../../../connect/jdbc/reference/sqlserverstatement-class.md) del objeto base de datos y recursos de JDBC inmediatamente en lugar de esperar a que se liberen automáticamente.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void close()  
```  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método de cierre especificado por el método de cierre en la interfaz java.sql.Statement.
