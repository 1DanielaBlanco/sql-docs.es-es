# <a name="getasciistream-method-sqlservernclob"></a>Método getAsciiStream (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el **NCLOB** valor designado por este **NClob** objeto como un flujo ASCII.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.sql.InputStream getAsciiStream()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto InputStream que contiene los datos NCLOB.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getAsciiStream especificado por el método getAsciiStream en la interfaz java.sql.SQLServerNClob.
