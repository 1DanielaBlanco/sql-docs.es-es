# <a name="setdescription-method-sqlserverdatasource"></a>Método setDescription (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece la descripción del origen de datos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void setDescription(java.lang.String description)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *Descripción*  
  
 A **cadena** que contiene la descripción.
