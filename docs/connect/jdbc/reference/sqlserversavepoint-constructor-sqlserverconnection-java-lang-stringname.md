# <a name="sqlserversavepoint-constructor-sqlserverconnection-javalangstringname"></a>Constructor SQLServerSavepoint (SQLServerConnection, java.lang.StringName)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Inicializa una nueva instancia de la [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) clase basada en la conexión especificada y el nombre.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public SQLServerSavepoint(SQLServerConnection con,  
                          java.lang.String sName)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *con*  
  
 A [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) objeto.  
  
 *sName*  
  
 A **cadena** que contiene el nombre del punto de retorno.
