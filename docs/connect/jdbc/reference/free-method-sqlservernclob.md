# <a name="free-method-sqlservernclob"></a>Método free (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Este método libera el objeto NCLOB y los recursos que contiene.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void free()  
```  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método libre especificado por el método libre en la interfaz java.sql.NClob.
