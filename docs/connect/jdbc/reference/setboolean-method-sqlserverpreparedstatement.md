# <a name="setboolean-method-sqlserverpreparedstatement"></a>setBoolean (método) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado el determinado **booleano** valor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final void setBoolean(int n,  
                             boolean x)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *n*  
  
 Un **int** que indica el número de parámetro.  
  
 *x*  
  
 A **booleano** valor, ya sea **true** o **false**.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método setboolean es especificado por el método setboolean en la interfaz java.sql.PreparedStatement.
