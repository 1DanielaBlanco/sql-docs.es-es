# <a name="issearchable-method-sqlserverresultsetmetadata"></a>Método isSearchable (SQLServerResultSetMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Indica si la columna designada se puede utilizar en una cláusula WHERE de SQL.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public boolean isSearchable(int column)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *columna*  
  
 Un **int** que indica el índice de columna.  
  
## <a name="return-value"></a>Valor devuelto  
 **True** si la columna de la columna se puede utilizar en una cláusula WHERE. De lo contrario, se devuelve el valor **False**.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método isSearchable especificado por el método isSearchable en la interfaz java.sql.ResultSetMetaData.
