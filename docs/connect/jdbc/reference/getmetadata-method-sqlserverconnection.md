# <a name="getmetadata-method-sqlserverconnection"></a>getMetaData (método) (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera un [SQLServerDatabaseMetaData](../../../connect/jdbc/reference/sqlserverdatabasemetadata-class.md) objeto que contiene metadatos acerca de la base de datos a la que se [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) objeto representa una conexión.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.sql.DatabaseMetaData getMetaData()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 El objeto DatabaseMetaData.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getMetaData especificado por el método getMetaData de la interfaz java.sql.Connection.
