# <a name="getcharacterstream-method-sqlserverclob"></a>Método getCharacterStream (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el **Clob** valor designado por el objeto java.SQL.DatabaseMetaData como un objeto java.io.Reader.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Description|  
|----------|-----------------|  
|[Método getCharacterStream &#40; &#41;](../../../connect/jdbc/reference/getcharacterstream-method.md)|Devuelve los datos de Clob como un objeto java.io.Reader o como un flujo de caracteres.|  
|[getCharacterStream método &#40; long, long &#41;](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Devuelve los datos de Clob como un objeto java.io.Reader o como una secuencia de caracteres con la posición y longitud especificadas.|
