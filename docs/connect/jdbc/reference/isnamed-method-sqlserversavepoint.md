# <a name="isnamed-method-sqlserversavepoint"></a>Método isNamed (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Comprueba si se ha denominado el punto de retorno.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public boolean isNamed()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 **True** se ha denominado el punto de retorno. De lo contrario, se devuelve el valor **False**.
