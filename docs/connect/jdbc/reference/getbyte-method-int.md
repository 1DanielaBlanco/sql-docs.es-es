# <a name="getbyte-method-int"></a>Método getByte (int)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un **bytes** según el índice del parámetro de valor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public byte getByte(int index)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *index*  
  
 Un **int** que indica el índice del parámetro.  
  
## <a name="return-value"></a>Valor devuelto  
 A **bytes** valor.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getByte es especificado por el método getByte en la interfaz java.sql.CallableStatement.
