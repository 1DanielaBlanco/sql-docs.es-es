# <a name="getdatabaseproductname-method-sqlserverdatabasemetadata"></a>Método getDatabaseProductName (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el nombre de este producto de base de datos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.lang.String getDatabaseProductName()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto **String** que contiene el nombre del producto de la base de datos.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notas  
 Este método getDatabaseProductName especificado por el método getDatabaseProductName en la interfaz java.sql.DatabaseMetaData.
