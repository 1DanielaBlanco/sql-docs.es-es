# <a name="datetimeoffset-class"></a>Clase DateTimeOffset
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Este tipo representa un objeto de tipo microsoft.sql.DateTimeOffset y es el tipo de Java que se corresponde con el [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] **datetimeoffset** tipo de datos.  
  
 **Paquete:** microsoft.sql  
  
 **Extiende:** java.lang.Object  
  
 **Implementa:** java.lang.Comparable\<DateTimeOffset >, java.io.Serializable  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final class DateTimeOffset  
```
