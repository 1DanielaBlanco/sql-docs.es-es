# <a name="gettransactiontimeout-method-sqlserverxaresource"></a>Método getTransactionTimeout (SQLServerXAResource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Obtiene el valor de tiempo de espera de transacción actual establecido para este [SQLServerXAResource](../../../connect/jdbc/reference/sqlserverxaresource-class.md) objeto.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getTransactionTimeout()  
```  
  
## <a name="exceptions"></a>Excepciones  
 javax.transaction.xa.XAException  
  
## <a name="remarks"></a>Comentarios  
 Este método getTransactionTimeout especificado por el método getTransactionTimeout en la interfaz javax.transaction.xa.XAResource.
