# <a name="updatetimestamp-method-sqlserverresultset"></a>Método updateTimestamp (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Actualiza la columna designada con un valor de marca de tiempo.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[updateTimestamp (int, java.sql.Timestamp)](../../../connect/jdbc/reference/updatetimestamp-method-int-java-sql-timestamp.md)|Actualiza la columna designada con un valor timestamp según el índice de la columna.|  
|[updateTimestamp (java.lang.String, java.sql.Timestamp)](../../../connect/jdbc/reference/updatetimestamp-method-java-lang-string-java-sql-timestamp.md)|Actualiza la columna designada con un valor timestamp según el nombre de la columna.|
