# <a name="settrustmanagerclass-method-sqlserverdatasource"></a>setTrustManagerClass (método) (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el valor de cadena de la propiedad de conexión TrustManagerClass.
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void setTrustManagerClass(java.lang.String trustManagerClass)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *trustManagerClass*  
  
 A **cadena** que contiene el nombre de clase completo de un javax.net.ssl.TrustManager personalizado.
