# <a name="setshort-method-sqlserverpreparedstatement"></a>setShort (método) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado el determinado **corto** valor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final void setShort(int index,  
                           short x)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *index*  
  
 Un **int** que indica el número de parámetro.  
  
 *x*  
  
 A **corto** valor.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método setShort especificado por el método setShort en la interfaz java.sql.PreparedStatement.
