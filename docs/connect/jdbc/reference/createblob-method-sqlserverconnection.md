# <a name="createblob-method-sqlserverconnection"></a>Método createBlob (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crea un objeto Blob sin ningún dato.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.sql.Blob createBlob()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto Blob.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método createBlob especificado por el método createBlob en la interfaz java.sql.Connection.  
  
 Este método no es necesario [SQLServerBlob Constructor &#40; SQLServerConnection, byte &#41; ](../../../connect/jdbc/reference/sqlserverblob-constructor-sqlserverconnection-byte.md).
