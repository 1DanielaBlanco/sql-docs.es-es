# <a name="getmaxcolumnsintable-method-sqlserverdatabasemetadata"></a>Método getMaxColumnsInTable (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el número máximo de columnas de que esta base de datos permite en una tabla.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getMaxColumnsInTable()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un **int** que indica el número máximo de columnas permitidas.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getMaxColumnsInTable especificado por el método getMaxColumnsInTable en la interfaz java.sql.DatabaseMetaData.
