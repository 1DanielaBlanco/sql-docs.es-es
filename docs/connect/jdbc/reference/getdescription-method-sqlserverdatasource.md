# <a name="getdescription-method-sqlserverdatasource"></a>Método getDescription (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve una descripción del origen de datos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public java.lang.String getDescription()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 A **cadena** que contiene la descripción del origen de datos o null si se establece ningún valor.
