# <a name="length-method-sqlservernclob"></a>Método length (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el número de caracteres de la **NClob**.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 La longitud de la **NClob** en número de caracteres.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método de longitud se especifica mediante el método de longitud de la interfaz java.sql.NClob.
