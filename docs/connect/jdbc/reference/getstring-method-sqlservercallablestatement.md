# <a name="getstring-method-sqlservercallablestatement"></a>Método getString (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un **cadena** en el lenguaje de programación Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Description|  
|----------|-----------------|  
|[getString (int)](../../../connect/jdbc/reference/getstring-method-int.md)|Recupera el valor del parámetro designado como un **cadena** en el lenguaje según el índice de parámetro de programación Java.|  
|[getString (java.lang.String)](../../../connect/jdbc/reference/getstring-method-java-lang-string.md)|Recupera el valor del parámetro designado como un **cadena** en el lenguaje de según el nombre del parámetro de programación Java.|
