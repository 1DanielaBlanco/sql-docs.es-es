# <a name="getfetchsize-method-sqlserverresultset"></a>getFetchSize (método) (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el tamaño de captura para este [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) objeto.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getFetchSize()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un **int** que indica el tamaño de captura actual.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getFetchSize especificado por el método getFetchSize en la interfaz java.sql.ResultSet.
