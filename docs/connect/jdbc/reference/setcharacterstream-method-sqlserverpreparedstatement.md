# <a name="setcharacterstream-method-sqlserverpreparedstatement"></a>Método setCharacterStream (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado para el objeto java.io.Reader determinado.

|Nombre|Description|  
|----------|-----------------|  
|[setCharacterStream método &#40; int, java.io.Reader &#41;](../../../connect/jdbc/reference/setcharacterstream-method-int-java-io-reader.md)|Establece el parámetro designado para el objeto java.io.Reader especificado.|  
|[setCharacterStream método &#40; int, java.io.Reader, int &#41;](../../../connect/jdbc/reference/setcharacterstream-method-int-java-io-reader-int.md)|Establece el parámetro designado en el objeto java.io.Reader especificado, que es el número especificado de caracteres de longitud.|  
|[setCharacterStream método &#40; int, java.io.Reader, long &#41;](../../../connect/jdbc/reference/setcharacterstream-method-int-java-io-reader-long.md)|Establece el parámetro designado en el objeto java.io.Reader especificado, que es el número especificado de caracteres de longitud.|
