# <a name="getreference-method-sqlserverxadatasource"></a>Método getReference (SQLServerXADataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve una referencia a este [SQLServerXADataSource](../../../connect/jdbc/reference/sqlserverxadatasource-class.md) objeto.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public javax.naming.Reference getReference()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un objeto de referencia.  
  
## <a name="remarks"></a>Comentarios  
 Este método getReference especificado por el getReference (método) en la interfaz javax.naming.Referenceable.
