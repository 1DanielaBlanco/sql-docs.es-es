# <a name="getmaxbinaryliterallength-method-sqlserverdatabasemetadata"></a>Método getMaxBinaryLiteralLength (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el número máximo de caracteres hexadecimales que permite que esta base de datos en un archivo binario en línea literal.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getMaxBinaryLiteralLength()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un **int** que indica el número máximo de caracteres hexadecimales.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getMaxBinaryLiteralLength especificado por el método getMaxBinaryLiteralLength en la interfaz java.sql.DatabaseMetaData.
