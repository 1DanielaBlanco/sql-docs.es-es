# <a name="getdriverminorversion-method-sqlserverdatabasemetadata"></a>Método getDriverMinorVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el número de versión secundaria de este controlador JDBC.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public int getDriverMinorVersion()  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Un **int** que indica la versión secundaria del controlador JDBC.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getDriverMinorVersion especificado por el método getDriverMinorVersion en la interfaz java.sql.DatabaseMetaData.
