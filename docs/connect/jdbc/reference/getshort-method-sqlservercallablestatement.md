# <a name="getshort-method-sqlservercallablestatement"></a>Método getShort (SQLServerCallableStatement)

  Recupera el valor del parámetro designado como un **breve** en el lenguaje de programación Java.

|----------|-----------------|  
|[getShort (int)](../../../connect/jdbc/reference/getshort-method-int.md)|Recupera el valor del parámetro designado como un **breve** en el lenguaje según el índice de parámetro de programación Java.|  
|[getShort (java.lang.String)](../../../connect/jdbc/reference/getshort-method-java-lang-string.md)|Recupera el valor del parámetro designado como un **breve** en el lenguaje de según el nombre del parámetro de programación Java.|
