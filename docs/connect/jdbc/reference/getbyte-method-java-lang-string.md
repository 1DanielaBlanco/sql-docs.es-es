# <a name="getbyte-method-javalangstring"></a>Método getByte (java.lang.String)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un **bytes** según el nombre del parámetro de valor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public byte getByte(java.lang.String sCol)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *sCol*  
  
 A **cadena** que contiene el nombre del parámetro.  
  
## <a name="return-value"></a>Valor devuelto  
 A **bytes** valor.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método getByte es especificado por el método getByte en la interfaz java.sql.CallableStatement.
