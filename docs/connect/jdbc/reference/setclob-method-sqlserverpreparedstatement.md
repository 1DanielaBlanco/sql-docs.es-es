# <a name="setclob-method-sqlserverpreparedstatement"></a>Método setClob (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado en un objeto java.sql.Clob.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Description|  
|----------|-----------------|  
|[Método setClob &#40;int, java.io.Reader&#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader.md)|Establece el parámetro designado en un objeto java.sql.Clob.|  
|[Método setClob &#40;int, java.sql.Clob&#41;](../../../connect/jdbc/reference/setclob-method-int-java-sql-clob.md)|Establece el parámetro designado para el objeto java.SQL.DatabaseMetaData determinado.|  
|[Método setClob &#40;int, java.io.Reader, long&#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader-long.md)|Establece el parámetro designado en un objeto java.SQL.DatabaseMetaData, que es el número especificado de caracteres de longitud.|  
  
## <a name="remarks"></a>Comentarios  
 Ningún comportamiento del controlador del servidor. Para obtener detalles vea las especificaciones de la API de JSE 6.
