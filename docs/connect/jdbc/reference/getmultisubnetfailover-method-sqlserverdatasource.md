# <a name="getmultisubnetfailover-method-sqlserverdatasource"></a>getMultiSubnetFailover (método) (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve el valor de la **multiSubnetFailover** propiedad de conexión.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
public boolean getMultiSubnetFailover();  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Devuelve true o false, dependiendo de la configuración actual de la propiedad de conexión.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener más información sobre la **multiSubnetFailover** propiedad de conexión, consulte [estableciendo las propiedades de conexión](../../../connect/jdbc/setting-the-connection-properties.md).
