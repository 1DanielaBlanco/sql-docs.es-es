# <a name="setinstancename-method-sqlserverdatasource"></a>Método setInstanceName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el nombre de instancia de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void setInstanceName(java.lang.String instanceName)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *instanceName*  
  
 Un **cadena** que contiene el nombre de instancia.
