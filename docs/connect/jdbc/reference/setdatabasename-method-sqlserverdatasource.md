# <a name="setdatabasename-method-sqlserverdatasource"></a>Método setDatabaseName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el nombre de la base de datos a la que se va a efectuar la conexión.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public void setDatabaseName(java.lang.String databaseName)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *databaseName*  
  
 A **cadena** que contiene el nombre de la base de datos.
