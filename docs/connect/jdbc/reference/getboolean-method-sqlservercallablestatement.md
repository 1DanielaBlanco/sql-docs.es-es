# <a name="getboolean-method-sqlservercallablestatement"></a>Método getBoolean (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Description|  
|----------|-----------------|  
|[getBoolean (int)](../../../connect/jdbc/reference/getboolean-method-int.md)|Recupera el valor del parámetro designado como un **booleano** según el índice del parámetro de valor.|
