# <a name="getasciistream-method-sqlservercallablestatement"></a>Método getAsciiStream (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera el valor del parámetro designado como un flujo de caracteres **ASCII**.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nombre|Descripción|  
|----------|-----------------|  
|[getAsciiStream &#40;int&#41;](../../../connect/jdbc/reference/getasciistream-int.md)|Recupera el valor del parámetro designado como un flujo de caracteres **ASCII** según el índice del parámetro.|  
|[getAsciiStream &#40;java.lang.String&#41;](../../../connect/jdbc/reference/getasciistream-java-lang-string.md)|Recupera el valor del parámetro designado como un flujo de caracteres **ASCII** según el nombre del parámetro.|
