# <a name="sqlserverclob-class"></a>Clase SQLServerClob
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Representa un objeto binario grande de caracteres (CLOB).  
  
 **Paquete:** com.microsoft.sqlserver.jdbc  
  
 **Extiende:** java.lang.Object  
  
 **Implementa:** Java.SQL.DatabaseMetaData, java.io.Serializable  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public class SQLServerClob  
```  
  
## <a name="remarks"></a>Comentarios  
 Un CLOB está almacenado en [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] como un tipo de datos TEXT o NTEXT.
