# <a name="setblob-method-sqlserverpreparedstatement"></a>setBlob (método) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Establece el parámetro designado para el objeto Blob determinado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
public final void setBlob(int i,  
                          java.sql.Blob x)  
```  
  
#### <a name="parameters"></a>Parámetros  
 *i*  
  
 Un **int** que indica el número de parámetro.  
  
 *x*  
  
 Un objeto Blob.  
  
## <a name="exceptions"></a>Excepciones  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Comentarios  
 Este método setBlob especificado por el método setBlob en la interfaz java.sql.PreparedStatement.
