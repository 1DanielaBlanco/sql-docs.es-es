# <a name="geturl-method-sqlservercallablestatement"></a>Método getURL (SQLServerCallableStatement)

  Recupera el valor del parámetro designado como un objeto de dirección URL en el lenguaje de programación Java.

|----------|-----------------|  
|[getURL (int)](../../../connect/jdbc/reference/geturl-method-int.md)|Recupera el valor del parámetro designado como un objeto de dirección URL en el lenguaje de programación Java según el índice del parámetro.|
