# <a name="getapplicationintent-method-sqlserverdatasource"></a>Método getApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Devuelve el valor de la **applicationIntent** propiedad de conexión.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
public java.lang.String getApplicationIntent();  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Devuelve el valor de la **applicationIntent** propiedad de conexión.  
  
## <a name="remarks"></a>Notas  
 Para obtener más información sobre la **applicationIntent** propiedad de conexión, consulte [estableciendo las propiedades de conexión](../../../connect/jdbc/setting-the-connection-properties.md).
