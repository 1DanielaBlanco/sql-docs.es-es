[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  Ibcpsession2, interfaz es una extensión IBCPSession que proporciona una función miembro que es una alternativa a llamar a ibcpsession:: BCPColFmt para cada columna.
