# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  Ibcpsession2, interfaz es una extensión IBCPSession que proporciona una función miembro que es una alternativa a llamar a ibcpsession:: BCPColFmt para cada columna.  IBCPSession2 hereda de IBCPSession y agrega un nuevo método: [ibcpsession2:: Bcpsetbulkmode](../../oledb/ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
