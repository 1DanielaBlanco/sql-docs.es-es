# Crear, eliminar o modificar una carpeta (portal web)
  Puede crear carpetas para organizar y administrar los elementos que publica en un servidor de informes. La creación de carpetas puede ayudar a los usuarios a buscar informes de su interés. Para administradores de contenido, las carpetas proporcionan un marco para aplicar permisos. Puede crear asignaciones de roles en carpetas concretas para restringir el acceso a los informes que se están desarrollando o que no se deberían distribuir de manera amplia.  
  
#### Para crear una carpeta  
  
1.  Inicie el [Administrador de informes &#40;Modo nativo de SSRS&#41;](../Topic/Report%20Manager%20%20\(SSRS%20Native%20Mode\).md).  
  
2.  En el portal web, seleccione **Carpeta** en el menú desplegable **Nuevo**. O, si quiere crear una carpeta en una carpeta existente, vaya a la carpeta en cuestión y seleccione **Carpeta** en el menú desplegable **Nuevo**.  
  
     Se abre el cuadro de diálogo **Crear una carpeta nueva**.  
  
3.  Escriba el nombre de la carpeta. Puede incluir espacios, pero no caracteres reservados que se utilicen para la codificación de direcciones URL: ; ? : @ & = + , $ / * \< > |. No se puede escribir una serie de nombres de carpetas para crear varias carpetas al mismo tiempo.  
  
4.  Seleccione **Crear**.  
  
#### Para modificar o eliminar una carpeta  
  
1.  En el portal web, vaya a la página **Inicio** y localice el elemento que quiera modificar.  
  
2.  Seleccione **...** en la carpeta que quiera.  
  
3.  En el menú contextual, seleccione **Administrar**. Se abre la página Propiedades.  
  
4.  Modifique las propiedades de la carpeta de las siguientes maneras:  
  
    -   Para modificar el texto que se muestra sobre la carpeta, escriba un nombre o una descripción.  
  
    -   Para mostrar la carpeta en la vista predeterminada en la página **Inicio**, desactive **Ocultar en la vista de mosaico**.  
  
5.  O bien, haga clic en **Eliminar**para quitar la carpeta y su contenido.  
  
6.  Haga clic en **Aplicar** para guardar los cambios.
