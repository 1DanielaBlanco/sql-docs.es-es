# Cuadro de di&#225;logo Secciones del informe (Generador de informes)
  Al abrir un informe, el Generador de informes comprueba que el informe contiene solo un elemento **\<Section>** del lenguaje RDL (Report Definition Language). No se permite abrir un informe con varios elementos **\<Section>**.  
  
 Aunque se puede generar una definición de informe con varias secciones mediante programación, no se admite la edición del informe en un entorno de creación de informes.  
  
 Para identificar el formato de definición de informe actual de un informe, un servidor de informes o un proyecto, o bien para el entorno de creación de informes, vea [Buscar la versión del esquema de definición de informe &#40;SSRS&#41;](../../reporting-services/reports/find-the-report-definition-schema-version-ssrs.md).  
  
 Haga clic en **Aceptar** para continuar.
