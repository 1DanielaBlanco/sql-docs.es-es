# <a name="configurationsetting-property---pathname"></a>Propiedad ConfigurationSetting - PathName
  Especifica la ruta de instalación de una instancia del servidor de informes. Solo lectura.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## <a name="property-values"></a>Valores de propiedad  
 Un objeto **String** que representa la ruta de instalación.
