# Propiedad SendUsingSMTPServer (MSReportServer_ConfigurationSetting de WMI)
  Especifica si el elemento **SendUsing** de la configuración del correo electrónico para el servidor de informes se ha establecido en **true**.  
  
## Sintaxis  
  
```vb  
Public Dim SendUsingSMTPServer As Boolean  
```  
  
```csharp  
public Boolean SendUsingSMTPServer;  
```  
  
## Valores de propiedad  
 Un objeto **booleano** de solo lectura.
