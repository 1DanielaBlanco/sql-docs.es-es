# Propiedad SenderEmailAddress (MSReportServer_ConfigurationSetting de WMI)
  Obtiene la dirección utilizada para enviar el correo electrónico desde el servidor de informes. Solo lectura.  
  
## Sintaxis  
  
```vb  
Public Dim SenderEmailAddress As String  
```  
  
```csharp  
public string SenderEmailAddress;  
```  
  
## Valores de propiedad  
 Un objeto **String** de solo lectura que representa la dirección de correo electrónico usada por el servidor de informes.
