# Propiedad DatabaseServerName (MSReportServer_ConfigurationSetting de WMI)
  Especifica el nombre del servidor en el que está instalada la base de datos del servidor de informes. Lectura/escritura  
  
## Sintaxis  
  
```vb  
Public Dim DatabaseServerName As String  
```  
  
```csharp  
public string DatabaseServerName;  
```  
  
## Valores de propiedad  
 Un objeto **String** que representa el nombre del servidor.
