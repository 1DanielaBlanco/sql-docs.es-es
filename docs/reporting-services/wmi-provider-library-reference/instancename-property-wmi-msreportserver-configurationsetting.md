# Propiedad InstanceName (MSReportServer_ConfigurationSetting de WMI)
  Especifica el nombre de una instancia del servidor de informes en un equipo específico. Solo lectura.  
  
## Sintaxis  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## Valor de la propiedad  
 Un objeto **String** que representa el nombre de la instancia del servidor de informes.
