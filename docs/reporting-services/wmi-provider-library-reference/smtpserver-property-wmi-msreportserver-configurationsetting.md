# Propiedad SMTPServer (MSReportServer_ConfigurationSetting de WMI)
  Obtiene la propiedad de servidor SMTP del archivo de configuración del servidor de informes. Solo lectura.  
  
## Sintaxis  
  
```vb  
Public Dim SMTPServer As String  
```  
  
```csharp  
public string SMTPServer;  
```  
  
## Valores de propiedad  
 Objeto **String** de solo lectura que contiene el valor de la propiedad **SMTPServer** del archivo RSReportServer.config.
