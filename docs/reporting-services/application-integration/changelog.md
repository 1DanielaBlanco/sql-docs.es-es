# <a name="changelog-for-the-report-viewer-controls"></a>Registro de cambios para los controles del Visor de informes

Este registro de cambios realiza un seguimiento del lanzamiento de los controles del Visor de informes WinForms y WebForms.

## <a name="15900148"></a>15.900.148
Con esta actualización
 - Corrección de un error que impide que los informes sin parámetros se carguen a través de Server.LoadReportDefinition

El control del Visor de informes [WebForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Webforms/150.900.148)
 - Compatibilidad con la inserción en las páginas de derecha a izquierda (páginas que cambian el flujo de texto mediante la propiedad css *direction: rtl;*).
 - Compatibilidad con la personalización de texto del cuadro de diálogo de impresión a través de la interfaz de localización *IReportViewerMessages5*.
 - Compatibilidad de accesibilidad mejorada.

El control del Visor de informes [WinForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Winforms/150.900.148)
 - Corrección para la impresión cuando se ejecuta una aplicación en modo con valores altos de PPP.
