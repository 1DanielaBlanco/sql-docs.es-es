# <a name="removing-a-data-processing-extension"></a>Quitar una extensión de procesamiento de datos
  Para quitar una extensión de procesamiento de datos, basta con quitar el **extensión** (elemento) para la extensión de procesamiento de datos del archivo de configuración. Si ha realizado las entradas de un servidor de informes, así como el Diseñador de informes, quite el **extensión** elemento desde archivos RSReportServer.config y RSReportDesigner.config. Después de quitar la información de configuración, la extensión de procesamiento de datos ya no estará disponible en el componente.
