# Crear un informe de tabla b&#225;sico (Tutorial de SSRS)
Con este tutorial, aprenderá a usar el Diseñador de informes para crear un informe de tabla básico basado en la base de datos **[!INCLUDE[ssSampleDBAdventureworks2014_md](../includes/sssampledbadventureworks2014-md.md)]**. También puede crear informes con el Generador de informes o con el Asistente para informes. En este tutorial, creará un proyecto de informes, configurará la información de conexión, definirá una consulta, agregará una región de datos de tabla, agrupará y calculará los totales de varios campos, y obtendrá una vista previa del informe.  
  
## Requisitos  
Para usar este tutorial, debe tener el software siguiente instalado en el sistema:  
  
-   [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)] motor de base de datos.  
  
-   [!INCLUDE[ssRSCurrent](../includes/ssrscurrent-md.md)] en modo nativo.  
  
-   Las base de datos [!INCLUDE[ssSampleDBAdventureworks2014_md](../includes/sssampledbadventureworks2014-md.md)].  Para obtener más información, consulte [Bases de datos de ejemplo Adventure Works 2014)](https://msftdbprodsamples.codeplex.com/releases/view/125550).  
  
 -   [SQL Server Data Tools](https://msdn.microsoft.com/library/mt204009.aspx) con los componentes de "SQL Server Reporting Services" instalados para que tenga el Diseñador de informes.    
  
También debe disponer de permisos de solo lectura para recuperar datos de la base de datos [!INCLUDE[ssSampleDBAdventureworks2014_md](../includes/sssampledbadventureworks2014-md.md)].

**Tiempo estimado para completar este tutorial:** 30 minutos.
