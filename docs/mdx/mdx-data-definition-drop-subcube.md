# <a name="mdx-data-definition---drop-subcube"></a>Definición de datos MDX - SUBCUBO de COLOCACIÓN
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  Quita un subcubo especificado y revierte la definición del cubo o subcubo definido previamente con el nombre especificado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
DROP SUBCUBE Subcube_Name   
```  
  
## <a name="arguments"></a>Argumentos  
 *Subcube_Name*  
 Expresión de cadena válida que proporciona un nombre de subcubo.
