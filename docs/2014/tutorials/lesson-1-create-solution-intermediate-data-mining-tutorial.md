# <a name="lesson-1-creating-the-intermediate-data-mining-solution-intermediate-data-mining-tutorial"></a>Lección 1: Crear la solución de minería de datos intermedia (Tutorial intermedio de minería de datos)
  En el tutorial básico de minería de datos, ha creado un [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] proyecto que contiene una solución de minería de datos simple basada en la nueva [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)] base de datos.  
  
 Para este tutorial, creará otro [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] proyecto mediante el uso de [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]. Creará un [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] origen de datos que usa [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)]y agregar varias vistas del origen de datos nuevo a ese origen de datos, para admitir los escenarios y tipos de modelo.  
  
 Esta lección consta de la tarea siguiente:  
  
-   [Crear una solución y el origen de datos &#40;intermedio de Tutorial de minería de datos&#41;](../../2014/tutorials/creating-a-solution-and-data-source-intermediate-data-mining-tutorial.md)
