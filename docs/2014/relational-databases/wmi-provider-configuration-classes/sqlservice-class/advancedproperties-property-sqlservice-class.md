# <a name="advancedproperties-property-sqlservice-class"></a>Propiedad AdvancedProperties (clase SqlService)
  Obtiene una matriz de referencias de objeto que contienen las propiedades avanzadas del objeto `SqlService`.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.AdvancedProperties [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Objeto de la [clase SqlService](sqlservice-class.md) que representa el servicio.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Una matriz de [clase SqlServiceAdvancedProperty](../sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) objetos que contienen las propiedades avanzadas para la `SqlService` objeto.
