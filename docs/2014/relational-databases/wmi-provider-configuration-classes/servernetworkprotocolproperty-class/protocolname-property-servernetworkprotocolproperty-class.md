  Obtiene el nombre del protocolo que posee la propiedad a la que se hace referencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*
