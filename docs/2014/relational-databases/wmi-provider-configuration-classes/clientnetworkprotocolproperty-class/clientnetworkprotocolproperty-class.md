# <a name="clientnetworkprotocolproperty-class"></a>Clase ClientNetworkProtocolProperty
  La [clase ClientNetworkProtocolProperty](clientnetworkprotocolproperty-class.md) representa propiedades de un protocolo de red del cliente.  
  
 Con la [clase ClientNetworkProtocolProperty](clientnetworkprotocolproperty-class.md)se pueden realizar las siguientes tareas:  
  
-   Establecer un valor de índice para seleccionar una propiedad del protocolo de red del cliente de la lista de propiedades.  
  
-   Comprobar el protocolo al que pertenece la propiedad actual.  
  
-   Comprobar el nombre, el tipo y el valor de la propiedad actual.  
  
-   Modificar los valores de cadena y los valores numéricos de la propiedad actual.  
  
-   Establecer marcas para la propiedad actual.
