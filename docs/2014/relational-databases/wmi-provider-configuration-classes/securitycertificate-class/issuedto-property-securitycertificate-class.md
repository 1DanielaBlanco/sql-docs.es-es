# <a name="issuedto-property-securitycertificate-class"></a>Propiedad IssuedTo (clase SecurityCertificate)
  Obtiene el objeto al que se emite el certificado de seguridad.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.IssuedTo [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Objeto de la [clase SecurityCertificate](securitycertificate-class.md) que representa un certificado de seguridad.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el objeto al que se emite el certificado de seguridad.
