# <a name="flagvalue-property-serversettingsgeneralflag-class"></a>Propiedad FlagValue (clase ServerSettingsGeneralFlag)
  Obtiene el valor de la marca.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.FlagValue [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Objeto de la [clase ServerSettingsGeneralFlag](serversettingsgeneralflag-class.md) que representa una marca general para la configuración del servidor.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor booleano que especifica el valor de la marca.
