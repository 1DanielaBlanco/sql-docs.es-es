# <a name="ipaddressname-property-servernetworkprotocolipaddress-class"></a>Propiedad IpAddressName (clase ServerNetworkProtocolIPAddress)
  Obtiene el nombre de la dirección IP.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.IpAddressName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 A [clase ServerNetworkProtocolIPAdress](servernetworkprotocolipaddress-class.md) objeto que representa una dirección IP para el protocolo de red en la instancia de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de cadena que especifica el nombre de la dirección IP.
