# <a name="numberofflags-property-serversettings-class"></a>Propiedad NumberOfFlags (clase ServerSettings)
  Obtiene el número de marcas generales asociadas a la instancia de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.NumberOfFlags [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 Objeto de la [clase ServerSettings](serversettings-class.md) que representa la configuración del servidor en una instancia de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Valor de `uint32` que especifica el número de marcas generales asociadas a la instancia de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
