# <a name="protocolname-property-servernetworkprotocol-class"></a>Propiedad ProtocolName (clase ServerNetworkProtocol)
  Obtiene el nombre del protocolo de red del servidor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Partes  
 *object*  
 A [clase ServerNetworkProtocol](servernetworkprotocol-class.md) objeto que representa el protocolo de red utilizado por la instancia de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Cadena que especifica el nombre del protocolo de red del servidor.
