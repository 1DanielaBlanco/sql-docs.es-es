|Nombre del atributo|Tipo|Descripción|  
|--------------------|----------|-----------------|  
|SQL_SOPT_SS_PARAM_FOCUS|SQLUINTEGER|Para obtener más información acerca de SQL_SS_PARAM_FOCUS, vea [SQLSetStmtAttr](../native-client-odbc-api/sqlsetstmtattr.md).|
