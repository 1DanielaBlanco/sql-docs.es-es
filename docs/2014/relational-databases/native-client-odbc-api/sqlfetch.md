# <a name="sqlfetch"></a>SQLFetch
  SQLFetch admite habían mejorado las características de fecha y hora.  
  
 Para obtener información acerca de cómo se convierten los valores de columna de resultados de tipos de fecha y hora, vea [conversiones de SQL a C](../native-client-odbc-date-time/datetime-data-type-conversions-from-sql-to-c.md).  
  
## <a name="sqlfetch-support-for-large-clr-udts"></a>SQLFetch admite UDT CLR grandes  
 `SQLFetch` admite tipos CLR definidos por el usuario (UDT) grandes. Para obtener más información, consulte [Large CLR User-Defined tipos &#40;ODBC&#41;](../native-client/odbc/large-clr-user-defined-types-odbc.md).
