# <a name="manage-logins-and-passwords-in-replication"></a>Administrar inicios de sesión y contraseñas en la replicación
  Cuando configure la replicación, especifique los inicios de sesión y las contraseñas para los agentes de replicación. Después de configurar la replicación, puede cambiar los inicios de sesión y las contraseñas. Para más información, consulte [View and Modify Replication Security Settings](view-and-modify-replication-security-settings.md). Si cambia la contraseña de una cuenta usada por un agente de replicación, ejecute [sp_changereplicationserverpasswords &#40;Transact-SQL&#41;](/sql/relational-databases/system-stored-procedures/sp-changereplicationserverpasswords-transact-sql).
