# <a name="administration-replication"></a>Administración (replicación)
  Esta sección contiene información sobre cómo administrar topologías de replicación. Se recomienda leer primero el tema de prácticas recomendadas y, a continuación, seguir los vínculos de ese tema a la información más detallada en esta y otras secciones.  
  
## <a name="in-this-section"></a>En esta sección  
 [Best Practices for Replication Administration](best-practices-for-replication-administration.md) (Prácticas recomendadas para la administración de replicación)  
 Describe diversas prácticas recomendadas e incluye información acerca de las copias de seguridad y restauración, la supervisión, el rendimiento y las tareas de mantenimiento.  
  
 [Preguntas más frecuentes para administradores de replicación](frequently-asked-questions-for-replication-administrators.md)  
 Aborda una amplia variedad de preguntas relacionadas con las tareas administrativas y el funcionamiento de la replicación.  
  
 [Mantener publicaciones](../publish/maintain-publications.md)  
 Describe cómo agregar y quitar artículos en las publicaciones, y proporciona una lista de cambios de propiedades que requieren la reinicialización o la generación de una instantánea nueva.  
  
 [Replication Agent Administration](../agents/replication-agent-administration.md) (Administración del Agente de replicación)  
 Proporciona información acerca de los agentes de replicación, los perfiles de agente y el uso de alertas.  
  
 [Hacer copias de seguridad y restaurar bases de datos replicadas](back-up-and-restore-replicated-databases.md)  
 Detalla las consideraciones para crear copias de seguridad y restaurar bases de datos replicadas, y estrategias para cada tipo de replicación.  
  
 [Validar datos replicados](../validate-replicated-data.md)  
 Proporciona información acerca de la validación de datos en los suscriptores para determinar si los datos coinciden con los del publicador.
