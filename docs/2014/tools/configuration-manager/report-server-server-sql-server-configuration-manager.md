  Éste es el servicio que admite [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)]. Para ver las propiedades del servicio, en el panel de detalles, haga clic con el botón derecho en el servicio y, después, haga clic en **Propiedades**.  
  
 Para obtener información acerca de [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)], vea los Libros en pantalla.
