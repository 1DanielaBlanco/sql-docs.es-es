# <a name="how-to-export-reports"></a>Cómo exportar informes
  El Visor de informes del Asesor de actualizaciones le permite exportar una versión plana del archivo de informe. Esto le permite trabajar con la información en una hoja de cálculo o un documento de procesador de textos, así como enviar la información a otras personas por correo electrónico.  
  
### <a name="to-export-a-report"></a>Para exportar un informe  
  
1.  En el Visor de informes, muestre el informe que desee exportar. Para obtener instrucciones, consulte [Cómo: ver un informe del Asesor de actualización](../../../2014/sql-server/install/how-to-view-an-upgrade-advisor-report.md).  
  
2.  Para filtrar el informe, seleccione un valor en el **filtrar por** cuadro.  
  
3.  Haga clic en **Exportar informe**.  
  
4.  En el **archivo de exportación de informes** cuadro de diálogo, escriba un nombre de archivo, elegir un destino y, a continuación, haga clic en **Aceptar**. El archivo se guardará en formato CSV (valores separado por comas). Puede utilizar Microsoft Excel o aplicaciones similares para abrir el archivo. Si desea obtener los datos en otro formato, puede copiar la información de Excel en Microsoft Word o en otras aplicaciones.
