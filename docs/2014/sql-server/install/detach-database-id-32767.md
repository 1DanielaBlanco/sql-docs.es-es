# <a name="detach-database-id-32767"></a>Desasociar la base de datos ID 32767
  El Asesor de actualizaciones detectó una base de datos con la base de datos ID 32767. En [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]y versiones posteriores, este identificador de base de datos está reservado. Separe la base de datos antes de la actualización. Puede asociar la base de datos después de la actualización.
