# <a name="cluster-security-policy"></a>Directiva de seguridad de clúster
  Utilice la página Directiva de seguridad de clúster para configurar la directiva de seguridad de la instancia del clúster de conmutación por error.  
  
## <a name="options"></a>Opciones  
 Especifique grupos de dominio global o local para servicios de clúster. Todos los permisos de recursos se controlan mediante grupos de nivel de dominio que incluyen cuentas de servicio de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] como miembros de grupo. Para obtener más información sobre la funcionalidad de los identificadores de seguridad de servicio (SID) de [!INCLUDE[nextref_longhorn](../../includes/nextref-longhorn-md.md)], vea [Configurar los permisos y las cuentas de servicio de Windows](../../database-engine/configure-windows/configure-windows-service-accounts-and-permissions.md).
