# <a name="ready-to-complete-image"></a>Listo para completar la imagen
  La página **Listo para completar la imagen** proporciona un resumen de los valores de configuración que se aplican a la configuración de la instancia preparada.  
  
## <a name="options"></a>Opciones  
 En esta página dispone de las siguientes opciones:  
  
-   Para continuar con la instalación, haga clic en **Instalar**.  
  
-   Para cambiar la configuración de la instalación, haga clic en **Atrás**.  
  
-   Para salir de la instalación, haga clic en **Cancelar**.
