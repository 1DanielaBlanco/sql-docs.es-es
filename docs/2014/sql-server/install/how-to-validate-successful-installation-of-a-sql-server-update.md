# <a name="how-to-validate-successful-installation-of-a-sql-server-update"></a>Validar una instalación correcta de una actualización de SQL Server
  Para validar la instalación correcta de una actualización de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]:  
  
-   Compruebe que los servicios de [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] están en línea.  
  
-   Compruebe la información de versión devuelta por el @@version consulta.
