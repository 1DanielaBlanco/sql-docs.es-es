# <a name="proxy-editor-and--add-principal"></a>Editor de proxy y Agregar entidad de seguridad
  Utilice esta página para conceder a las entidades de seguridad del servidor acceso a una cuenta de proxy del Agente [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="options"></a>Opciones  
 **Tipo de entidad de seguridad**  
 Tipo de entidad de seguridad que se muestra.  
  
 **Entidades de seguridad disponibles**  
 Presenta una lista con las entidades de seguridad del tipo seleccionado.  
  
 **Nombre**  
 Seleccione el nombre de la entidad de seguridad a la que desea conceder acceso o elija **Nombre** para conceder acceso a todas las entidades de seguridad de la lista.
