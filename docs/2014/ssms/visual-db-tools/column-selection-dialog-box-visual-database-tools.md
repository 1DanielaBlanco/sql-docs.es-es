# <a name="column-selection-dialog-box-visual-database-tools"></a>Selección de columnas (cuadro de diálogo, Visual Database Tools)
  Permite cambiar la vista Personalizada de las tablas en el diagrama de base de datos. La vista Personalizada solo muestra las propiedades de columna identificadas por el usuario.  
  
 Este cuadro de diálogo aparece cuando se hace clic con el botón derecho en una tabla y después se elige **Modificar vista personalizada** en el menú contextual.  
  
## <a name="options"></a>Opciones  
 **Columnas disponibles**  
 Muestra todas las columnas existentes en la tabla seleccionada de la base de datos. Las propiedades de la tabla de base de datos y el tipo de base de datos determinan las columnas que se van a mostrar aquí. Resalte la columna que desee mostrar y utilice los botones de flecha para mover las columnas al cuadro **Columnas seleccionadas** .  
  
 **Columnas seleccionadas**  
 Muestra las propiedades de columna definidas actualmente para la vista personalizada. Utilice los botones de flecha para agregar y quitar propiedades de columna en la lista Columnas seleccionadas.  
  
 **Controles de flecha**  
 Use los botones de flecha arriba y abajo para mover las columnas resaltadas hacia arriba o hacia abajo en la lista **Columnas seleccionadas** .  
  
 Utilice las flechas > y >> para agregar una o todas las propiedades a la vista personalizada.  
  
 Utilice las flechas < y << para quitar una o todas las propiedades de la vista personalizada.  
  
 **Guardar como predeterminado**  
 Reemplaza la vista personalizada predeterminada por las columnas seleccionadas en este cuadro de diálogo. Si no está activada, la selección de columnas especificada en el cuadro de diálogo solo se aplicará a la tabla seleccionada en el diagrama de base de datos.  
  
 **Aceptar**  
 Guarda la vista personalizada.  
  
 **Cancelar**  
 Cancela la modificación de la vista personalizada.
