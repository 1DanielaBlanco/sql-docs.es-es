# <a name="delete-a-database-diagram-visual-database-tools"></a>Eliminar un diagrama de base de datos (Visual Database Tools)
  Cuando ya no necesite un diagrama de base de datos, puede eliminarlo.  
  
### <a name="to-delete-a-database-diagram"></a>Para eliminar un diagrama de base de datos  
  
1.  En el Explorador de objetos, expanda la carpeta **Diagramas de base de datos** .  
  
2.  Haga clic con el botón secundario en el diagrama de base de datos que desea eliminar.  
  
3.  Elija **Eliminar** en el menú contextual.  
  
4.  Un mensaje le pedirá que confirme la eliminación. Elija **Sí**.  
  
 El diagrama de base de datos se elimina de la base de datos. Al eliminar un diagrama de base de datos, no se eliminan las tablas del diagrama.
