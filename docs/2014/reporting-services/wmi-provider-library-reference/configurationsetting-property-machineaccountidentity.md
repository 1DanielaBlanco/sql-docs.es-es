# <a name="machineaccountidentity-property-wmi"></a>Propiedad MachineAccountIdentity (WMI)
  Obtiene la identidad de la cuenta del equipo en el que está instalado el servidor de informes.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>Valores de propiedad  
 Un `String` valor que contiene la identidad de la cuenta de máquina del equipo que está instalado el servidor de informes.
