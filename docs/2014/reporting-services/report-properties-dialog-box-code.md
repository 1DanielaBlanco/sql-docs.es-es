# <a name="report-properties-dialog-box-code"></a>Propiedades del informe (cuadro de diálogo), Código
  Seleccione **Código**en el cuadro de diálogo **Propiedades del informe** para definir el código personalizado que usarán las expresiones en el informe.  
  
## <a name="options"></a>Opciones  
 **Código personalizado**  
 Escriba los métodos que se van a utilizar en el informe. El código se debe escribir en [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[vbprvb](../includes/vbprvb-md.md)].
