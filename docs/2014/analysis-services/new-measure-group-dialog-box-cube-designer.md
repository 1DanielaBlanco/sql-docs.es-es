# <a name="new-measure-group-dialog-box-cube-designer"></a>Nuevo grupo de medida (cuadro de diálogo del Diseñador de cubos)
  Use el cuadro de diálogo **Nuevo grupo de medida** en [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] para agregar un nuevo grupo de medida a un cubo en el Diseñador de cubos. Puede mostrar el cuadro de diálogo **Nuevo grupo de medida** de uno de estos modos:  
  
-   Haciendo clic en **Nuevo grupo de medida** en el panel **Barra de herramientas** de la pestaña **Estructura de cubo** del Diseñador de cubos.  
  
-   Haciendo clic con el botón derecho en una medida o en un grupo de medida en el panel **Medidas** de la pestaña **Estructura de cubo** del Diseñador de cubos y seleccionando **Nuevo grupo de medida** en el menú contextual.  
  
## <a name="options"></a>Opciones  
 **Seleccione una tabla en la vista del origen de datos**  
 Seleccione la tabla de hechos para el nuevo grupo de medida. Las tablas no se pueden usar más de una vez como tablas de hechos dentro del mismo cubo.
