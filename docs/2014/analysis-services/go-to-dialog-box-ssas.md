# <a name="go-to-dialog-box-ssas"></a>Ir a, cuadro de diálogo (SSAS)
  El cuadro de diálogo **Ir a** le permite trasladarse a una columna concreta de una tabla.  
  
## <a name="uielement-list"></a>Lista de UIElement  
  
|||  
|-|-|  
|Término|Definición|  
|**Vete a**|Seleccione una columna en la lista **Ir a** y haga clic en **Aceptar**.|
