# <a name="details-ssas"></a>Detalles (SSAS)
  Esta página del **Asistente para la importación de tablas** le permite ver información sobre una operación de importación de datos que concluyó correctamente o en la que se produjo un error. Para tener acceso al asistente desde [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)], en el menú **Modelo** , haga clic en **Importar desde el origen de datos**.
