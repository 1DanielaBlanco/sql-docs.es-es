# <a name="viewing-models-data-mining-add-ins-for-office"></a>Ver modelos (Complementos de minería de datos para Office)
  Los Complementos de minería de datos para Excel proporcionan diversas formas de explorar los modelos analíticos que ha creado:  
  
-   Use los gráficos en Excel generados por cada asistente.  
  
     [Examinar modelos en Excel &#40;complementos de minería de datos de SQL Server&#41;](browsing-models-in-excel-sql-server-data-mining-add-ins.md)  
  
-   Use las Formas de minería de datos para Visio para crear una presentación interactiva.  
  
     [Ver modelos de minería de datos en Visio &#40;complementos de minería de datos&#41;](viewing-data-mining-models-in-visio-data-mining-add-ins.md)  
  
-   Use la **documento modelo** Asistente para generar un resumen de los metadatos del modelo y el contenido del modelo.  
  
     [Documentar modelos de minería de datos &#40;datos complementos de minería de datos para Excel&#41;](documenting-mining-models-data-mining-add-ins-for-excel.md)
