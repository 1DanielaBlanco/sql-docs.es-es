# <a name="command-events-event-category"></a>eventos de comandos (categoría de eventos)
  La categoría de eventos Eventos de comandos tiene las clases de eventos que se describen en la siguiente tabla.  
  
|Clase de eventos|Identificador del evento|Descripción|  
|-----------------|--------------|-----------------|  
|Command Begin|15|Recopila todos los eventos de inicio de comando desde el inicio del seguimiento.|  
|Command End|16|Recopila todos los eventos finales de comando desde el inicio del seguimiento.|  
  
 Para obtener información sobre las columnas asociadas a cada una de las clases de evento de Eventos de comandos, vea [Command Events Data Columns](command-events-data-columns.md).
