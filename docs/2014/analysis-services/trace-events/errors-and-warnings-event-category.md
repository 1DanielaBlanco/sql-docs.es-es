# <a name="errors-and-warnings-event-category"></a>Errores y advertencias (categoría de eventos)
  La categoría de eventos Errores y advertencias tiene las clases de eventos que se describen en la siguiente tabla.  
  
|Clase de eventos|Identificador del evento|Descripción|  
|-----------------|--------------|-----------------|  
|Error|17|Registra todos los nuevos eventos de error desde el inicio del seguimiento.|  
  
 Para obtener información acerca de las columnas asociadas a cada una de las clases de eventos de Errores y advertencias, vea [Errors and Warnings Events Data Columns](errors-and-warnings-events-data-columns.md).
