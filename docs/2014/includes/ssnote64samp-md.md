SQL Server Data Tools no se admite en Itanium\-equipos basados en. Sin embargo, la compatibilidad con SQL Server Data Tools está disponible para x64\-equipos basados en. Si se han implementado las bases de datos de ejemplo de SQL Server en un Itanium\-equipo basado en, utilice BI Development Studio en cualquier un x86\-basado o x64\-en función de equipo para modificar y ejecutar los ejemplos.
