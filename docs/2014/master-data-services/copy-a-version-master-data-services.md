# <a name="copy-a-version-master-data-services"></a>Copiar una versión (Master Data Services)
  En [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], copie una versión del modelo para crear una nueva versión del mismo.  
  
## <a name="prerequisites"></a>Requisitos previos  
 Para realizar este procedimiento:  
  
-   Debe disponer del permiso para tener acceso al área funcional de **Administración de versiones** .  
  
-   Debe ser administrador de modelo. Para obtener más información, vea [Administrators &#40;Master Data Services&#41;](administrators-master-data-services.md).  
  
### <a name="to-copy-a-version"></a>Para copiar una versión  
  
1.  En [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)], haga clic en **Administración de versiones**.  
  
2.  En la página **Administrar versiones** , seleccione la fila de la versión que desea copiar.  
  
    > [!NOTE]  
    >  Según el valor de una opción de [!INCLUDE[ssMDScfgmgr](../includes/ssmdscfgmgr-md.md)], es posible que pueda copiar únicamente las versiones con el estado **Confirmado** . Para obtener más información, vea [Configuración del sistema &#40;Master Data Services&#41;](../../2014/master-data-services/system-settings-master-data-services.md).  
  
3.  Haga clic en **Copiar**.  
  
4.  En el cuadro de diálogo de confirmación, haga clic en **Aceptar**.
