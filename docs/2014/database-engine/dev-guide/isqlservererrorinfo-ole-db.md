# <a name="isqlservererrorinfo-ole-db"></a>ISQLServerErrorInfo (OLE DB)
  El [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] proveedor OLE DB de Native Client define la **ISQLServerErrorInfo** interfaz de errores. Esta interfaz devuelve los detalles de una [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] error, incluso la gravedad y estado.  
  
## <a name="in-this-section"></a>En esta sección  
  
|Método|Descripción|  
|------------|-----------------|  
|[ISQLServerErrorInfo:: GetErrorInfo &#40;OLE DB&#41;](../../relational-databases/native-client-ole-db-interfaces/isqlservererrorinfo-geterrorinfo-ole-db.md)|Devuelve un puntero a un [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] proveedor OLE DB de Native Client estructura SSERRORINFO que contiene la [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] detalles del error.|
