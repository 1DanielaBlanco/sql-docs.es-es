# <a name="run-sql-server-profiler"></a>Ejecutar SQL Server Profiler
  Para ejecutar [!INCLUDE[ssSqlProfiler](../includes/sssqlprofiler-md.md)], en el menú **Inicio** , elija **Todos los programas**, **Microsoft SQL Server 2014**, **Herramientas de rendimiento**y, a continuación, haga clic en **SQL Server Profiler**.
