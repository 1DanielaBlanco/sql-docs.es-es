# <a name="copy-or-export-target-data"></a>Copiar o exportar datos de destino
  Puede copiar los datos de destino y también puede exportarlos a un archivo .CSV.  
  
> [!NOTE]  
>  No se pueden copiar o exportar los datos de destino para el destino **ring_buffer** .  
  
## <a name="to-copy-target-data"></a>Para copiar los datos de destino  
  
-   En la ventana de los datos de destino, seleccione las filas que desee copiar, haga clic con el botón secundario y, a continuación, seleccione **Copiar**.  
  
## <a name="to-export-target-data-to-a-csv-file"></a>Para exportar datos de destino a un archivo .CSV  
  
1.  En la ventana de los datos de destino, seleccione las filas que desee exportar, haga clic con el botón secundario y, a continuación, seleccione **Exportar**.  
  
2.  En el cuadro de diálogo **Guardar como** , especifique la carpeta y el nombre del archivo.  
  
3.  Haga clic en **Aceptar** para guardar los datos de destino como un archivo CSV.
