# <a name="annotations-master-data-services"></a>Anotaciones (Master Data Services)
  En [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], las anotaciones son comentarios que se escriben para proporcionar detalles sobre transacciones.  
  
 Puede agregar una anotación a una transacción para proporcionar información sobre por qué se realizó cierta acción. Debe seleccionar una transacción para agregar una anotación relacionada. Las anotaciones se muestran junto con las transacciones en:  
  
-   El área funcional del **Explorador** de la aplicación web [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] .  
  
-   El área funcional **Administración de versiones** de la aplicación web [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] .  
  
-   El [!INCLUDE[ssMDSXLS](../includes/ssmdsxls-md.md)]de MDS.
