# P&#225;gina Configuraci&#243;n del servidor de Master Data Services (Administrador de configuraci&#243;n)
  Utilice la página **Configuración del servidor** para ver información de configuración del equipo del servidor local.  
  
## Configuración del servidor  
  
|Nombre del control|Description|  
|------------------|-----------------|  
|**Internet Information Services**|Muestra la versión de Internet Information Services (IIS) que se instala en el equipo local. Se requiere la versión 7.0 o posterior de IIS para utilizar el Administrador de configuración de Master Data Services.|  
|**Versión del producto MDS**|Muestra la versión del producto de [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)].|
