# <a name="change-sets-and-approval-mds-add-in-for-excel"></a>Change Sets and Approval (complemento de MDS para Excel)
  En el complemento [!INCLUDE[ssMDSshort](../../includes/ssmdsshort-md.md)] para Excel, puede guardar los cambios pendientes de una entidad en un conjunto de cambios. Puede aplicar un conjunto de cambios existente con el fin de ver y modificar los cambios pendientes y, además, puede confirmar un conjunto de cambios existente.  
  
 Si la entidad requiere la aprobación de los cambios, debe guardar los cambios pendientes en un conjunto de cambios y enviarlo para recibir la aprobación. Para obtener más información, consulte [Aprobación necesaria &#40;Master Data Services&#41;](../../master-data-services/approval-required-master-data-services.md)
